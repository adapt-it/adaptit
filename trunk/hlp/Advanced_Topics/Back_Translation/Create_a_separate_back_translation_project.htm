Create a separate back translation project

The creation of a separate back translation project, involves two basic procedures:

  1. Export the translation (target) texts from the current adaptation project. See the Export target text procedure.

  2. Create a new Adapt It project as a back translation project in which the translation texts exported above are used as the source text. See Create a project for the steps to create a project.

 Note

  • When you create the project, be sure to use the language name of the exported adaptation text in the Source edit box.

  • In the Target edit box type the name of the language to be used in the back translation (the language of the consultant).

  • As you establish settings, remember that the target texts from the original project are now the source texts of the new back translation project, and you are adapting new target texts as back translations in the consultant’s language.

  • The original text in the source language is not available in this type of back translation project.
