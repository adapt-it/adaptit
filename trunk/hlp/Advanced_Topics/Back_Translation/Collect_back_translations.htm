Collect back translations

The Collect Back Translations… option of the Advanced menu can be used to create translations from existing adaptations or glosses. The command copies the text of the glosses or adaptations and places them into a back translation placeholder. This back translation becomes a starting point and you can edit it later to become well-formed back translations Adapt It stores back translations as within Adapt It documents.

To incorporate existing glosses or adaptations as back translations, or to create empty back translations, do the following:

  1. From the current project open a document.

  2. Determine if you want to collect a back translation for the entire document, only a portion of the document, or to create empty back translations.

    • If you wish to collect a back translation for the entire document, continue on to the next step.

    • Select the portion of the source for which you want to have a back translation.

    • If you wish to create empty back translation, determine if there are any current adaptations or glosses in the document. You can create an empty back translation only if either there is no adaptation or nor no glossing in the document.

  3. On the Advanced menu click Collect Back Translations…

    The Collect Text As Back translations dialog appears.

  4. Do one of the following:

    • To collect back translations from existing adaptation lines, select the Form back translations from the adaptation line's text option.

    • To collect back translations from any existing gloss lines, select the Form back translations from the gloss line's text option.

    • To create empty back translation placeholders, choose an option for which there are no entries:

      If there are no gloss entries in the document, select the Form back translations from the gloss lines' text option.

      If there are no adaptations in the document select the Form back translations from the adaptation line's text option.

  5. Click OK.

    Adapt It scans the document and gathers together sections of the chosen option, adds a back translation marker (\bt) to each section, and stores each section as filtered material.

 Warning

  • Collecting a back translation removes any previous back translation.

  • If text is currently selected, the collection takes place only within the selection range. Collection ends either at the end of the selection, or when Adapt It encounters the next back translation.

 Note

  • Back translations are indicated on Adapt It's main window by small (usually green) wedge shaped icons. You may click on these icons to view and edit the filtered material stored in those locations. See View Filtered Material.

  • You can add back translation text and edit it later, if you select the Collect Back Translations… option and choose to create empty back translation placeholders.

  • A gray wedge shaped icon marks an empty back translation.
