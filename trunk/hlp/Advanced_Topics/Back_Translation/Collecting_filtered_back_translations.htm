Collecting filtered back translations

Adapt It can store back translations inside its documents and manage them in the form of . Filtered material such as back translations are stored in the background and can only be seen by clicking on a green wedge. See View filtered material for more information.

Use the Collect Back Translations… command on the Advanced menu to incorporate back translations into the Adapt It document. You can use this option to do any one of the following:

  • Collect the current adaptation information as a back translation. In other words, copy the target text into the filtered back translation.

  • Collect the current gloss information as a back translation. In other words, copy the gloss test into the filtered back translations. You may have created these glosses directly or transformed them from another project. See Transform adaptations into glosses

  • Create empty back translation placeholders throughout a document.

For more information see Collect back translations.

Once you have incorporated a back translation into an Adapt It document, you can view and edit it. Also, when you export translation (target) texts, Adapt It exports the back translations as well. This process is called propagation. Thus, if you use the translation text as the source text in another project, the back translation is already available. See Propagation of filtered information.
