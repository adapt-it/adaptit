Create a gloss line literal back translation

To create a literal word by word back translation using the gloss line, do the following steps:

  1. Create the glossing entries

  2. Collect the back translation from the glossing line using Collect back translations.

 Note

  • All glosses are stored in a glossing knowledge base, not in the adaptations knowledge base.

  • Merging source words into phrases and unmerging of phrases are not permitted while in glossing mode, but you can make a merge by switching to adaptation mode. In glossing mode, any merges done in adaptation mode are treated as if they were a single source text word.

  • Retranslations and inserted placeholders are not permitted while in glossing mode.
