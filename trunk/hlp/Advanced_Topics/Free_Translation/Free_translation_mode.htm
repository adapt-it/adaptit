Free translation mode

Free translation mode is an advanced feature of Adapt It. Free translation mode can be turned on or off at any time. In free translation mode, Adapt It is able to compose and intelligently display free translations in a line below the strips in its main window - keeping the free translation close to the text to which it applies.

In free translation mode, a compose bar and navigation buttons are visible above the main window. These are used to create the free translation, move about, and adjust the span of the free translation.

See Create a free translation for more information and detailed steps.
