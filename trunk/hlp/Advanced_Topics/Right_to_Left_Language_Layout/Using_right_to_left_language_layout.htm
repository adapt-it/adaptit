Using right to left language layout

While the default language direction for Adapt It is left to right, Adapt It Unicode can also accommodate languages that read right to left both as Source or Target languages. That is the both languages can read right to left, but also one language can read left to right and the other right to left.

There are two general ways of controlling the direction of the language.

  • Use the Fonts tab or dialog to establish that either or languages read right to left.

  • Use the Layout window option of the .

    This option is a toggle that changes the name depending upon the current orientation of the window:

    • If the window is currently left to right the option reads: Layout Window Right To Left CTRL+1.

    • If the window is currently right to left the option reads: Layout Window Left To Right CTRL+1.

    Ctrl+1 can be used instead of the menu option.

 Note

  • We cannot fully discuss right to left languages in these Helps. However, the system font and encoding functionalities on which Adapt It Unicode relies render the Phrase box text in the correct direction. This is true even if you do not use the checkboxes provided in the Fonts tab. Setting the source text’s layout as discussed above is your responsibility. This option controls the order of the piles with in each adaptation line. You can change the direction at any time.
