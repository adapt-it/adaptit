Viewing and editing filtered material

Back translations, free translations, and Adapt It notes are stored within Adapt It documents as filtered material. Filtered material is represented in Adapt It's main window by small wedge shaped icons (usually green).

The filtered material icon indicates the point in the text where the filtered material is hidden. When the icon applies to a filtered back translation or free translation, that material applies to the text immediately following the location of the wedge.

In a document containing filtered material, you can view that material by doing the following steps:

  1. Click on the wedge at the location you wish to view.

    The View Filtered Material dialog appears.

  2. If the Marker list box at the left contains more than one standard format marker, click on the desired marker to see the associated text in the middle edit box.

    The text appears in the middle box, the Marker Description text changes to describe the type of marker, and the caption of the Remove button changes to reflect the marker material that would be removed.

  3. If desired, edit the text in the middle edit box.

    If you make changes in the text, the OK command button changes to Save Changes.

  4. To remove the current filtered material, click the Remove --- command button. The caption of the button varies according to the marker selected.

  5. To close the dialog do one of the following:

    • Click OK.

    • Click Save Changes to close the dialog and save your changes.

    • Click Cancel to close the dialog and discard any changes you made.
