Set filtering preferences

To set the preferences for how Adapt It filters (hides) information during adaptation, do the following steps:

  1. From the current project open a document.

  2. Select Preferences… from the Edit menu.

    The Preferences dialog appears (see Preferences dialog)

  3. Select the Filtering tab.

    The Filtering page appears (see Filtering tab).

  4. In either of the top two lists, scroll the list and find the marker you want to change.

  5. Select the marker and its description by clicking anywhere on its line in the list.

    If the marker's box is already checked for filtering the button to the right of the list reads Include for Adapting. If the marker's box is not checked for filtering the button to the right reads Filter Out.

  6. Do one of the following:

    • Click on Include for Adapting to cause the text associated with the marker to be made available for adapting in Adapt It's main window.

    • Click on Filter Out to cause the text associated with the marker to be filtered out (hidden from view) in Adapt It's main window.

  7. Repeat Steps 5 and 6 above for any other markers you wish to filter out or include for adapting.

  8. Click OK to make any filtering changes effective, or Cancel to quit without making any changes.

 Warning

  • Make any needed switch of marker sets before making any Filtering changes. See USFM tab and Set USFM Preferences.

  • If you Filter Out (hide) a marker which previously had adaptations made for its associated text, those adaptations will be lost in all places where that marker appeared. Then, if you later change that marker's filter state again to Include for Adapting, you will have to re-adapt the text associated with that marker in all locations within the document where that marker reappears.

 Note

  • If you change the filtering status for any marker, Adapt It automatically rebuilds the document so the text is laid out to reflect the new status.
