Placement of filtered material in interlinear RTF exports

When exporting the current document as an interlinear RTF document file, you can control how back translations, free translations, and notes appear in the output. The Export Filter/Option button of the Export RTF Interlinear Document dialog opens the Export Filter/Options dialog.

The Special placement options for Back translations, Free Translations and Adapt It Notes exports box at the bottom of the dialog controls how these items will appear in the output file. This box includes three options:

  • Place Free Translations in a separate table row within the exported interlinear text (default).

  • Place Back Translations in separate table row within the exported interlinear text

  • Place Adapt It Notes in balloon text Comments in the margin

The option titles are self explanatory. For samples of exported interlinear text see Interlinear text (with free translation) and Interlinear text (with free and back translation)

 Note

  • If you unselect any of the options, the material for that option appears as footnotes.
