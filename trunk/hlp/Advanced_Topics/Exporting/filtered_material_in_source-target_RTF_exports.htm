Placement of filtered material in source or target RTF exports

When exporting source, or target texts to RTF files, you can control how back translations, free translations, and notes appear in the output document. The Export Filter/Option button of the Export Document dialog opens the Export Filter/Options dialog. See Export source text or Export target text on how to find the button.

The Special placement options for Back translations, Free Translations and Adapt It Notes exports box at the bottom of the dialog controls how these items will appear in the output file. This box includes three options:

  • Place Free Translations in double boxed paragraphs within the exported text (default)

  • Place Back Translations in single boxed paragraphs within the exported text

  • Place Adapt It Notes in balloon text Comments in the margin (default)

The option titles are self explanatory. For a sample of exported text see Example of exported source or target text including filtered material (RTF).

 Note

  • If you unselect any of the options, the material for that option appears as footnotes.
