Change Export Filter/Options for source or target texts in standard format (SFM)

This procedure allows you to control which standard format markers are included in an export of the source or target texts. The default is to Export all markers and associated text including filtered text.

If you want to filter out (omit) certain markers and associated text during export:

  1. Click the Filter out selected markers and associated text from the exported document option.

    Adapt It makes the list box available for editing.

  2. Select a marker you want to change for the export.

  3. Depending on the buttons available, do one of the following:

    • Click Filter Out to omit the marker and its associated text from the exported text.

    • Click Include in Export to keep the marker and its associated text in the exported text.

    • Click Undo to restore any changes you have made.

  4. Click OK to accept your choice of options.

    Adapt It returns you to the main Export Document dialog where you can start (or cancel) the actual export process.

 Warning

  • Filtering of information from exports is not the same thing as filtering of information in the adaptation process. This topic is about filtering (omitting) markers and text information from exports. If you want to hide certain information during the adaptation process see Filtering of Information.

 Note

  • The special placement options in the lower part of the dialog are not available for SFM exports.
