Edit a note

To edit an existing note, do the following steps:

  1. From the current project open the document which has one or more notes you want to edit.

    The location of each note stored in the document is marked by a pair of icons that look like this:

  2. Click on the yellow note icon to display the note.

    The Note dialog opens with the note displayed in the edit box.

  3. To edit the note, click anywhere within the contents of the note in the edit box and make changes as desired.

  4. Click on the OK button to save any changes you made to the note and to close the dialog. If you click on Cancel the dialog also closes, but Adapt It does not save any changes you made before closing the dialog.

 Tip

  • From the Note dialog, you can edit the note, delete it or locate and view other notes using the buttons at the bottom of the Note dialog. See delete a note, and navigate through notes.
