View a note

You can view any consultant notes, advisor notes, or exegetical notes that you or someone else has created in an Adapt It document. To view a note that was previously created in an Adapt It document, do the following steps:

  1. From the current project open the document which has one or more notes you want to view.

    The location of each note stored in the document is marked by a pair of icons that look like this:

  2. Click on the yellow note icon.

    The Note dialog opens with the note displayed in the edit box. You can edit it, delete it or locate and view other notes using the buttons at the bottom of the Note dialog. See edit a note, delete a note and navigate through notes.

  3. Click on the OK button to save any changes you made to the note and to close the dialog. If you click on Cancel the dialog also closes, but Adapt It does not save any changes you made before closing the dialog.
