Delete a note

To delete an existing note from an Adapt It document, do the following steps:

  1. From the current project open the document which has one or more notes you want to delete.

    The location of each note stored in the document is marked by a pair of icons that look like this:

  2. Click on the yellow note icon to display the note.

    The Note dialog opens with the note displayed in the edit box. You can edit it, delete it or locate and view other notes using the buttons at the bottom of the Note dialog. See edit a note, and navigate through notes.

  3. To delete the entire note, click on the Delete button at the bottom of the dialog.

    Adapt It does not ask you to verify your action before deleting the note. The note is deleted as soon as you click on the Delete button, and the dialog immediately closes.

    The yellow note icon disappears from the location when you delete the note. The green wedge also disappears unless there is filtered information of another kind still there.

 Note

  • If you want to remove all Adapt It notes from the current document you may click on the button on the toolbar. Clicking on the button brings up the Note Deletion warning dialog.

    Click Yes to remove all of the notes. Click No to return to the document without deleting any notes.
