Accept changes without stopping

The Accept Changes Without Stopping option of the Tools menu can only be used if Consistent Changes (CC) has been loaded. See Load Consistent Changes.

Once you have loaded CC, Adapt It makes the option available.

The purpose of this option is to allow Adapt It and CC to work together to completely adapt a document with little or no human input. In other words, when you choose this option Adapt It runs through the entire adaptation making adaptations in the following priorities:

  • Adaptation for the word or phrase in the knowledge base

  • The source text word or phrase as possibly changed by the CC rules

The only time the adaptation process stops is when the knowledge base contains two or more adaptations for the same word. At that point the translator must choose an adaptation. The process then continues automatically.

This tool is designed for projects where there are only minor differences between the languages and a very strong set of CC rules have been developed.

 Warning

  • Accept Changes Without Stopping should be only be used when the following are true:

    • The languages are highly related

    • The CC rules are very strong in that the consistent changes are regular, well defined and apply to a high proportion of the text to be adapted.

  • The use of CC can introduce a number of incorrect adaptations. Therefore, you need to carefully edit the adaptations after its use.

  • You need to balance the improved speed in adaptation when CC is used against the effort to edit incorrect adaptations.
