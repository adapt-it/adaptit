Use Consistent Changes

The Use Consistent Changes…option of the Tools menu can only be used if Consistent Changes (CC) has been loaded. See Load Consistent Changes.

Once you have loaded CC, Adapt It automatically selects the Use Consistent Changes option (places a tick mark in front of the option) and CC runs as described in Consistent Changes overview.

If you click on the option, the tick mark disappears and Adapt It does not use CC in its adaptation process.

Click the option again and Adapt It resumes using CC.

 Note

  • Using the Use Consistent Changes…option to turn off CC, does not remove the CC tables from the computers memory. Thus, if you turn CC back on, Adapt It continues to use the same tables.
