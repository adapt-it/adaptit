Load Consistent Changes…

To start Consistent Changes (CC) and load one or more changes tables, do the following:

  1. In the Tools menu select the Load Consistent Changes… option.

    Adapt It links itself to the CC software and opens the Get One Or More Consistent Change Tables dialog.

    There are four Table tabs allowing you to select up to four tables.

  2. In the list box for Table 1 tab, select the CC table you wish to use.

    To the right you will see the name of the table you have chosen.

  3. If you wish to use more than one table, select a new Table tab and repeat step 2 up to three times.

    In this way you can select up to four tables.

  4. If the table you want to use is not in the list, click the Browse To Find CC Table… command button.

    Adapt It opens the Locate The Consistent Change Table dialog. This dialog allows you to find and tell Adapt It where the file is located. (CC change table files must have a filename extension of “cct”.

    When you have found the file and selected it, click Open to return. Click Cancel to stop looking for a file.

    If you want to create a new change table, click .

  5. If you want to edit the selected change table, click Edit CC Table….

  6. If you do want not to include a change table previously selected, move to the Table tab where it is selected and click Select None.

    CC will no longer use the table in its processing. Tables selected in other tabs are not affected.

  7. To complete the process, do one of the following:

    • Click OK to accept the changes and return to the Main window.

    • Click Cancel to undo all of the changes and return to the Main window.

 Note

  • Loading Consistent Changes automatically turns on the CC program. You can turn off CC by clicking the Unload Consistent Changes option of the Tools menu so that the tick mark does not show. This removes the CC tables and stops the CC process. To stop CC in Adapt It without removing the tables click the Use Consistent Changes option.
