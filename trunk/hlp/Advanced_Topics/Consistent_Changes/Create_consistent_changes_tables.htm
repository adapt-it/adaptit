Create consistent changes tables

While you can use Consistent Changes (CC) tables created in other places, you can also build them directly. To do this you need to open CC by using the Load Consistent Changes option of the Tools menu and selecting the Create CC Table button.

To create a new table, do the following steps:

  1. Select the Load Consistent Changes option in the Tools menu.

    Adapt It displays the Get One Or More Consistent Change Tables dialog.

  2. Click the Create CC Table… button.

    Adapt It opens the Create CC Table dialog.

  3. Type the name of the new table into the text box.

    Type only the name of the table, Adapt It automatically adds a “cct” filename extension.

  4. Click OK (or Cancel to return to the Get One Or More Consistent Change Tables dialog).

    Adapt It opens the Edit Consistent Changes Table dialog. The text box normally will be empty; we show some rules as examples.

  5. Type your rules into the text box.

  6. When finished, Click OK (or Cancel to return to the Get One Or More Consistent Change Tables dialog).

 Tip

  • To learn about writing CC rules, read the document CC.doc in the Adapt It or Adapt It Unicode subfolders inside the Program Files folder of your computer.

 Note

  • You can open any of the CC tables that Adapt It knows about and add new rules or change any of the current rules. When you save the table, any changes you have made are immediately available for new adaptations. The changes are not applied to any previous adaptations.
