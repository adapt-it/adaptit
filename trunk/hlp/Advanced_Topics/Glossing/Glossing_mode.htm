Glossing mode

Glossing is an advanced feature of Adapt It. See Glossing overview.

You may view any glosses that have been entered for the current document by selecting the See Glossing item on the Advanced menu.

If you want to add or edit glosses to the current document, you must also check the Glossing check box on the Mode bar. When the Glossing box is checked Adapt It changes to glossing mode.

 Note

  • Glossing mode may be toggled on or off at any time using the Glossing checkbox.

  • While in glossing mode Adapt It displays the glossing line between the source text and target text and the phrase box will move only within the glossing line.

  • No source phrase merging or retranslations may be done while in glossing mode. Also placeholders cannot be entered while in glossing mode. If you need to merge source words, or do a retranslation, or insert a placeholder:

    • Turn glossing mode off

    • Do the desired operation

    • Turn the glossing mode back on

    • Resume glossing.

  • Whenever See Glossing is deselected on the Advanced menu, glossing mode is automatically turned off (and the box will need to be checked again the next time you want to use glossing mode).
