Exit

This section has only one option – Exit.

When you click on Exit, Adapt It displays a message box asking if you want to save the current document.

Choose one of the following options:

  • Yes

    Saves the document in its current state. In addition, Adapt It saves any changes made to the knowledge base in memory to the knowledge base on hard disk.

    The Adapt It session then closes.

  • No

    Discards any changes to the document and closes the Adapt It session. Changes to the knowledge base since the last time it was saved are also discarded.

  • Cancel

    Returns you to the document for further editing. No changes are discarded.
