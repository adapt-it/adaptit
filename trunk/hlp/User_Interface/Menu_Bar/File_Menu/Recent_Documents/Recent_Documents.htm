Recent Documents

Adapt It provides a method to quickly open documents that you have used recently. To open one of these documents:

  1. Click File on Menu bar.

    Adapt It displays the File menu. Just before the Exit option, the menu contains a list of up to nine documents.

  2. If the document you want is listed, click on it.

    Adapt It closes the current document, while asking you if you want to save any changes. If necessary, Adapt It also closes the current project. The program then opens the desired project and document.

 Note

  • Because of technical problems this technique only works for documents 2 through 9.
