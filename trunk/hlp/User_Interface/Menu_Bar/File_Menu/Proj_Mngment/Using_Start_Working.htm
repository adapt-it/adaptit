Using Start Working

The Start Working option operates in two modes depending upon whether or not a project is open:

  • If a project is open, Adapt It assumes that you want to open a document or create a new one in the current project. See Using Start Working for Documents.

  • If there is no project open, Adapt It assumes that you want to open a project or create a project. See Using Start Working for Projects

To use the Start Working option, do the following steps:

  1. Open the File menu.

  2. Click the Start Working option.

    Adapt It responds according to the status described above.
