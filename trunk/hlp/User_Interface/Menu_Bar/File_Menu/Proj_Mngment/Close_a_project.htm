Close project

This option closes the current project open document. To close the project, do the following steps:

  1. Click on the File menu.

    The menu opens. Close Project is available only if a project is open. (If a document is open in the Main window, a project is also open).

  2. Click Close Project.

    If a document is open, Adapt It presents a save dialog box. This box asks you if you want to save the changes to the current document.

  3. Click one of the following options.

    • Yes

      This option saves the current document and clears the Main window. It also saves any changes to the knowledge base. Adapt It then closes the project.

    • No

      This option closes the current document without saving any changes made since the document was last saved. (See Auto-Saving Tab for more information). Adapt It then closes the project.

    • Cancel

      Adapt It closes the dialog box and returns to the document in the Main window. Both the document and project remain open.

 Note

  • You can use Ctrl+J instead of the Close Project option.
