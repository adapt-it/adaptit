Print Preview

Print preview allows you to see how your print job will look like without using any paper. This option produces a window that shows you a picture of the document in the Print Preview window. This window is similar to the Main window.

To open the Print Preview window, do the following steps:

  1. Click on the File menu.

  2. Click the Print Preview option.

    Adapt It opens the Print Preview window.

    You now have seven options controlled by command buttons or special keys.

  3. Click on the desired command button:

    • Print command button (Alt+P) – this button opens the Print dialog box and allows you to print the document.

    • Next Page command button (Alt+N) – if the document contains more than one page moves the window to the next print page. This button does not work if the window is at the last page of the document.

    • Prev. Page command button (Alt+V) – if the document contains more than one page, and the window is not on the first page of the document moves the window to the previous page of the document.

    • Two page command button (Alt+T) / One page (Alt+O) – shows two pages of the document if there is enough room in the window.

    • Zoom in command button (Alt+I) – increases the size of the document to make it more readable. If the window was in Two page format, Adapt It changes the mode to One page format.

    • Zoom out command button (Alt+O) – decreases the size of the document so that you can see more of the page or switch to Two page format.

    • Close command button (Alt+C) -- closes the window.

Notes

  • Adapt It can show the document in three sizes controlled by the zoom commands.

  • You can also zoom by clicking on the document. When the zoom gets to the largest size, one more click changes the page to the smallest size.
