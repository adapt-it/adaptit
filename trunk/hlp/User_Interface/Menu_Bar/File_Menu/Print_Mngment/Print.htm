Print

This option allows you to print the document you are working on. To print, do the following steps:

  1. On the File menu click Print. (You may use Ctrl+P instead.)

    Adapt It opens the Print dialog box.

  2. Make any adjustments to the print job.

    See Using the Print dialog box.

  3. Click OK to print the document

    Adapt It prints the document to the chosen printer. You may see a message box showing the progress of the print job.

 Note

  • Generally, the most satisfactory way to print a document is to export it as interlinear text. You can then load the resulting file into a word processor and print it from there.

  • Printing from the Print command generates a report that is formatted using the piles and strips from the Main window. This gives you a printed copy of what is actually in the adaptation document. However, this report uses a lot of paper and is not as easy to read as a report from a word processor.

  • Normally only the source and target texts appear in the report. However, if Glossing mode is on, Print also includes the gloss entries. The See Glosses command on the Advanced menu turns on glossing.
