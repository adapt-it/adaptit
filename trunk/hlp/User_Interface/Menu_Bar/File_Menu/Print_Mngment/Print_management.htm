Print management

This second section of the File menu includes four items. The items may be available or not depending on what you are doing in Adapt It. If the item is dark you can use it. If the item is grayed out, you cannot use it at the current time.

  • Print

  • Print Preview

  • Print Setup

  • Page Setup

 Note

  • Print management is included at this point to be consistent with other MS Windows programs. However, if you need to print a document we recommend that you use an export function and save documents in txt or rtf format. You can then use a word processor to print the document. See Document management for export options

  • Printing directly out of Adapt It requires more white space than printing from a word processor; also, users tend to use larger font sizes. Consequently documents may require more paper than necessary. Word processors allow you better control over the printed matter.

  • Adapt It does not directly print special filter content such as free translations, back translation and notes. You need to export your documents as RTF to include these special features.
