Importing knowledge base entries

Adapt It can import either to the default adaptation knowledge or the glossing knowledge base. To import knowledge base entries, do the following steps:

  1. Decide which knowledge base you wish to import:

    If you are in adapting mode, to import:

    • The adaptation knowledge base, move on to the next step.

    • The glossing knowledge base, change to glossing mode and move to the next step

    If you are in glossing mode, to import:

    • The adaptation knowledge base, change to adapting mode by unselecting the See Glosses option of the Advanced menu. Move to the next step.

    • The glossing knowledge base, move to the next step.

  2. From the File menu, select the Import to knowledge base … option.

    Adapt It opens a Extend the knowledge base by importing dictionary records dialog box.

  3. Make sure that the folder that Adapt It selects contains the file you wish to import. If not, use the Look in combo box to select the location where the file is stored.

  4. Select the file you wish to import.

  5. Click Open to complete the process.

    Adapt It updates the knowledge base with the contents of import file.

 Note

  • Adapt It adds the entries in the import file to the knowledge base. If the knowledge base already contains the entry or part of the entry, Adapt It adds any new adaptations or glosses not already in the knowledge base.

  • Adapt It keeps all of the current entries in the knowledge base.

  • Each imported entry consists of two types of records. Each record starts with one of two standard format markers: \lx and\ge. The \lx record is used for source text information, while the \ge record is used for target text or gloss information. One entry consists of one \lx record and one or more \ge records. Each entry may be separated by a blank record. Adapt It ignores any other records that may be in the imported file.
