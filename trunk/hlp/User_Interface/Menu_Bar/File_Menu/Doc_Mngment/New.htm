New

This option allows you to add a new document to a project, but is available only when there is no current document visible in the . To open a new document, do the following steps:

  1. Make sure that there is no open document in the Main window. If there is an open document, close the document.

  2. Click on the File menu.

    The menu opens and the New item should be usable and not "grayed out."

  3. Click New.

    Adapt It opens the Input Text File for Adaptation dialog box.

    .

    This dialog box allows you search for the new source text file. You should have already prepared the file and know where to look.

    The file should be a plain text file, optionally containing USFM markup characters. Plain text files generally have a filename extension of "txt." However, files with other extensions can be used if they meet the requirement of being plain text. (Files with the extension of doc or rtf cannot be used.)

  4. Click on the filename.

    The file name appears in the File name text box.

  5. Click Open.

    Adapt It opens the Type Output Document Name dialog box. This dialog is where you give the adaptation document its name for storage on the hard disk.

  6. Type the name of the document into the text box and click OK.

    Adapt It closes the dialog box and uses the data from the Plain Text data file to build the adaptation document. It then lays out the adaptation document in the Main window.

    The document is now ready for adaptation.
