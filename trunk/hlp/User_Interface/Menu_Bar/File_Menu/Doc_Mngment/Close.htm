Close

This option closes any open document in the Main window. To close the document, do the following steps:

  1. Click on the File menu.

    The menu opens. Close is available only if there is a document in the Main window.

  2. Click Close.

    Adapt It opens a Save Changes message box . This box asks you if you want to save the changes to the document. Click the appropriate option.

  3. The Main window remains open for further work.
