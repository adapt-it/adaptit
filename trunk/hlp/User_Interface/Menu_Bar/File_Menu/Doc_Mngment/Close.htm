Close

This option closes any open document in the . To close the document, do the following steps:

  1. Click on the File menu.

    The menu opens. Close is available only if there is a document in the Main window.

  2. Click Close.

    Adapt It opens a save dialog box. This box asks you if you want to save the changes to the document.

  3. Click one of the following options.

    • Yes

      This option saves the current document and clears the Main window. It makes any changes permanent by updating the hard disk knowledge base.

    • No

      This option closes the current document without saving any changes made in the last 10 minutes. These changes are lost. (See Auto saving tabAuto saving tab for more information)

    • Cancel

      Adapt It does not do anything and returns to the document in the Main window.
