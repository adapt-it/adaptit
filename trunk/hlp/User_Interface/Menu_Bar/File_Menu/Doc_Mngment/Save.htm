Save

The Save option allows you to manually save the current document. To save a document, do the following steps:

  1. Click on the File menu.

    The menu opens and the Save item should be usable and not “grayed out.”

  2. Click Save.

    Adapt It saves the current version of the document, closes the file menu and returns to the document.

Notes

  • Saving the document means that all changes made in your current session are written to the hard drive.

  • Adapt It overwrites the previous version of the document. You cannot restore that version of the document.

  • It is a good idea to regularly save a document as you are working on it. If the computer should stop functioning for some reason, your work will not be lost.
