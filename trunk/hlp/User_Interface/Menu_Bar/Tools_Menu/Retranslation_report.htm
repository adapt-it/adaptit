Retranslation report

The Retranslation report option of the Tools menu allows you to generate reports of all the retranslations in a document or a project. A report may be of use to a translation consultant wishing to check your work for exegetical accuracy. Retranslations are a departure from the meaning of the source text and may introduce exegetical errors. This command generates a report so that retranslations can be found and checked.

Generate a retranslation report for a document

To generate a retranslation for a document, do the following steps:

  1. In the Tools menu select the Retranslation Report option.

    Adapt It displays a informational message stating that you can only create a report for the current document.

  2. Click Yes to continue.

    Adapt It displays a Save As dialog that allows you to specify the name of the output file.

  3. Click Save to create the file or Cancel to stop the report.

    Adapt It creates the file and returns to the Main window.

Generate a retranslation report for the project

To generate a retranslation report for the entire project, do the following steps:

  1. If a document is open, close it by using the Close option in the File menu.

    Adapt It clears the document from the Main window.

  2. In the Tools menu select the Retranslation Report option.

    Adapt It displays a Save As dialog that allows you to specify the name of the output file.

  3. Click Save to create the file or Cancel to stop the report.

    Adapt It displays the Remove Unwanted Files From the List dialog.

  4. Move any unwanted files to the right size list box.

  5. Click OK to continue or Cancel to stop the report.

    Adapt It reads through all of the remaining files and creates the report.
