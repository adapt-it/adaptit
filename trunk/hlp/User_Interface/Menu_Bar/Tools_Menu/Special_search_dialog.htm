Special search dialog

If you click the Special Search button, the special search mode of the Find dialog appears.

There are no text boxes, the Search In: options are not usable, and three Special Searches option buttons at the bottom left become active. In addition, the middle command button on the right is now Normal Search, and the Select a standard format marker: combo box appears. (You can use this combo box only if you have selected the Standard Format Marker option.)

The Special Searches options do the following:

  • Retranslation

    Adapt It searches for the next Retranslation it can find.

  • Placeholder

    Adapt It searches for the next Placeholder it can find.

  • Standard Format Marker

    Adapt It opens the Select a standard format marker combo box. You can then select one of the standard format markers (SFM) from the list. Adapt It searches for the next occurrence of that SFM.

After you have determined your search, do one of the following:

  • Click Find Next to activate the search.

  • Click Normal Search to return to the Find dialog.

  • Click Cancel to return to the Find dialog
