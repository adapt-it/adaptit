Split, Join, or Move document section

This section of the Tools menu, allows you to:

Split a document into smaller pieces

You can split large documents into several smaller documents by using Split Document… option. As Adapt It nears the end of a large document, its performance may slow as it makes automatic changes. Splitting a document in half and working in end half may improve performance. Splitting a document is also useful when preparing documents for use in low power machines such as the AlphaSmart Dana. Such machines may have difficulty working with large documents. See Split a document for more information.

Join two or more documents into a single document

In the case where you need to combine two or more documents into a larger document, you can use the Join Documents… option to accomplish this task. The documents that you join should be part of the same book. Be careful to ensure that you join the documents in the correct sequence so that the chapters will not be out of order.

Move a document from the Adaptations folder to a Book folder or vice versa

If you have decided to use Book folders mode, you may need to move documents from the Adaptations folder to the appropriate book folder. The Move A Document option allows you to move these documents.

Alternatively, if you have been working in Book folders mode and decide not to use this mode anymore; you will need to move the documents into the Adaptations folder. The Move A Document option allows you to do this as well. (You may find it helpful to rejoin small split off documents into one larger one and then move the large document. See Join documents.)
