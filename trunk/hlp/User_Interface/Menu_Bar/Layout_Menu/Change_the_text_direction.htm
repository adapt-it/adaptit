Change the text direction

To change the direction of the text, do one of the following options:

  • Press Ctrl+1

  • Use the Layout menu.

  1. On the Menu Bar, click Layout

    Adapt It opens the Layout menu

  2. Click the Layout Window Right To Left option.

Adapt It changes the direction of the text.

 Note

  • If you are already in Right to Left mode, the option reads Layout Window Left to Right.

  • Changing the layout direction does not produce any change in the relative order of characters within words.
