Edit menu

The Edit menu contains five sections:

  • Undo – this section contains only the Undo option.

  • Standard editing functions – these are three functions that are available in most computer applications.

  • Miscellaneous – this consists of the following three items:

    • Go to

    • Edit source text

    • Consistency checking

  • Moving Notes

  • Managing Preferences

To open the Edit menu:

  1. Click on the word Edit on the Menu bar.
