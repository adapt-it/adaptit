Go to

Go to allows you to move around in an Adapt It document using the Chapter and Verse SFM markers. To use Go to apply one of the follows techniques:

  1. Do one of the following

    • Press Ctrl+G

    • Open the Go to option in the Edit menu.

    Adapt it opens the Go to dialog box:

  2. Enter the desired chapter number into the Chapter list box.

  3. Enter the desired verse into the Verse list box.

  4. Click OK

    Adapt It positions the Phrase box at the first word or phrase of the desired chapter or verse.

 Note

  • Go to depends upon the SFM markers for chapter (\c) and verse (\v) being in the appropriate places in the original source text file. Adapt It shows these locations in the Navigation text. Without the markers, Adapt It cannot find the location and displays an error message.

  • If the desired location would place the phrase box at a retranslation, Adapt It positions the Phrase box in the location immediately before the retranslation.

  • If the document has no chapters, use zero as the chapter number.

  • If your document contains verse ranges, such as 1-4, instead of individual verse numbers, Adapt It can determine the range that contains an individual verse, such as 2. In this case, Adapt It would move the Phrase box to the first word of verse 1.
