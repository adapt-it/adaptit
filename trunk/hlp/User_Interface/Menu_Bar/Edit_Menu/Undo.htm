Undo

Most computer applications provide a quick way to correct a mistake that you have just made. This tool is called Undo. Undo reverses the action you have just done and puts everything back to where it was.

Adapt It supports Undo, but only for some operations in the Phrase box. Undo restores only your last editing operation, such as a BackSpace key press, Delete key press, Cut command, or Paste command.

To undo the most recent action, take one of the following actions:

  • Press Ctrl+Z

  • Use the Edit menu Undo option:

  1. Open the Edit menu in the Menu bar.

  2. Click the Undo option.

Adapt It reverses the previous action and restores everything to previous state.

 Note

  • We recognize that this level of Undo is rudimentary and Redo should be available as well. At some future point, we will be upgrading the Undo and Redo capability.
