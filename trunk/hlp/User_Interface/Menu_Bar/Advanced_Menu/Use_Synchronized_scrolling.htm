Use Synchronized scrolling

Synchronized scrolling is a feature that allows Adapt It to coordinate with other translation tools such as Paratext and Translator’s Workplace. This feature allows you to compare the text, at the same book, chapter and verse, as it appears in each of these tools.

This section of the Advanced menu contains two options:

  • Send Synchronized Scrolling messages

  • Receive Synchronized Scrolling messages

You can use either or both of the options. If you check the “Send” option and are coordinating with Paratext, as you move the Phrase box in Adapt It, the text in Paratext moves as well--scrolling itself to the same chapter and verse.

If you check the “Receive” option, then as you change to another verse, chapter or book in Paratext, Adapt It moves the Phrase box to the corresponding location in the target text.

 Note

  • When Adapt It receives scrolling commands from another application that require a change to a different book. Adapt It closes the current document and then searches for an adaptation document that contains the same book, chapter, and verse. If it finds a match it opens the document and places the Phrase box at that chapter and verse.
