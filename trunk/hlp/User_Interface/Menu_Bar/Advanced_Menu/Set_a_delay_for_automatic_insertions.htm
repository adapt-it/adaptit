Set a Delay for Automatic Insertions

As Adapt It does the adaptation process, it moves as rapidly as possible to insert translations from the knowledge base. At times the process moves so quickly it can be hard to follow. Therefore, Adapt It allows you to delay the process by up to 3 seconds for each insertion. To delay the insertion process, do the following steps:

  1. Open the Advanced menu

  2. Select the Delay… option.

    Adapt It opens the Set a Delay for Automatic Insertions dialog box.

    The default value is zero.

  3. Enter the desired delay time in hundredths of seconds.

    One hundred would delay the process one second as shown above. Two hundred would set the delay at two seconds. The maximum is 300 or three seconds.

  4. Click OK.

    Adapt It closes the dialog and displays the desired time delay in the Delay text box on the Mode bar.
