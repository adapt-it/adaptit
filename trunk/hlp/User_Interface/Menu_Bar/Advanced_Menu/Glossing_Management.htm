Glossing management

Glossing management controls whether or not you see glosses as you work and also allows you to control whether or not glosses can be entered into the document. See glossing mode for more information. To initiate the use of glossing:

  1. Open the Advanced menu.

  2. Select the See Glossing option.

    Adapt It makes the Glossing Uses Navigations Text’s Font option available.

 Note

  • The default font for glossing is the same as that for the target text. However the default font color is that of navigation text.

  • The default font color for glossing is the same as that of the navigation text.
