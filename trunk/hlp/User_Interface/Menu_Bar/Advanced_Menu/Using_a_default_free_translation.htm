Using a default free translation

The Free translation section of the Advanced menu contains two commands:

  • Use Target Text As Default Free Translation

  • Use Gloss Text As Default Free Translation.

You do not have to use either of these options, but can use only one of these options at a time.

If what already has been adapted or back translated is close to the way you want your free translation to be written, turn on the Use Target Text As Default Free Translation option. However, if the gloss text more closely resembles the free translation, choose Use Gloss Text as Default Free Translation.

To select either of the options: click once to turn it on, click it again to turn it off. If one of the options is already on and you click on the other, Adapt It turns off the first option. In other words, Adapt It does not allow you to select both of the options at the same time.

When in free translation mode and the phrase box moves to a new section, the compose bar's edit box will show you the collected text from the line you specify.

 Note

  • Adapt It can use the Gloss text even if Glossing mode is not on.
