Miscellaneous advanced options

The Miscellaneous advanced options section of the Advanced menu contains two options:

  • Transform adaptations into glosses…

    This option appears when all documents in the project are closed. You can use this feature only under the special circumstances describe in Transform adaptations into glosses.

  • Delay…

    This option allows you to slow down the speed at which Adapt It moves the Phrase box. See Set a delay for automatic insertions for more information.
