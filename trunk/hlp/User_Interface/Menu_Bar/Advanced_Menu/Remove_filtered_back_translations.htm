Remove filtered back translations

The Remove filtered back translation option of the Advanced menu allows you to remove all back translations from the document. When you choose this option, Adapt It displays a warning message:

  • Click Yes to remove all back translations from the document.

  • Click No to stop the process without removing any back translations.

 Warning

  • If you click Yes, all of the back translations you have entered are deleted and cannot be restored. You will have to retype any back translations you want to use.

  • To remove a single back translation see Viewing and editing filtered material.
