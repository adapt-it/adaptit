Using Book folders

To initiate the Book folders mode for a project, do the following steps:

  1. Open the Advanced menu on the Menu bar.

  2. Click the Storing Documents in Book Folders option.

    The following things happen:

    • If it has not done so already, Adapt It builds the 67 subfolders in the Adaptations folder in the project.

    • Book folder mode is turned on and Adapt It opens the Document dialog showing that the Book folder mode is on and lists the files in the Current book folder. You can identify the current folder by looking at the message that says “Documents are being saved to the Matthew folder.” (Matthew is the default folder; the message may name some other folder.)

  3. Do one of the following:

    • If the folder is acceptable move on to step 5.

    • If the folder is not acceptable, click Change Folder…

      Adapt It opens the Choose a Book Folder dialog. See Choose a book folder dialog for more information.

  4. Find the folder you want to use and click OK.

    Adapt It changes the folder and returns to the Document dialog.

  5. Do one of the following:

    • Select <New Document> and click Finish.

      Adapt It follows the procedure to Create a new document.

    • Select one of the current documents and click Finish.

      Adapt It opens the document for further work.

 Note

  • Adapt It does not automatically move files from the main Adaptations folder to any of the book folders when you turn Storing Documents in Book Folders on. It also does not automatically move files back to the Adaptations folder, if you turn Storing Documents in Book Folders off

  • If you are going to use Book folders, it will save you time and effort if you turn them on at the beginning of the project before doing any adaptations. Otherwise, you will have to move documents from the Adaptations folder into the appropriate book folder. See Move a document dialog.

  • The Move a document dialog allows you to move a document from the Adaptations folder to a book folder, and also to move a document from a book folder to the Adaptations folder.

  • You can only move a document when Storing Documents in Book Folders is on.
