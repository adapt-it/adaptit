Menus

The Menus toolbar follows the pattern of Windows applications and contains seven drop-down menus.

Five of these are standard Windows menus and there are two special Adapt It menus named Advanced and Layout. Each of the standard menus includes entries that relate only to Adapt It.

  • File

  • Edit

  • View menu

  • Tools menu

  • Advanced menu

  • Layout menu (Only in Adapt It Unicode)

  • Help menu
