Prevent source text from being copied to the phrase box

As Adapt It performs the adaptation process, if it does not find an adaptation in the knowledge base, it copies the source text into the Phrase box. You can change this behavior. By doing the following steps:

  1. Open the View menu in the Menu bar.

  2. Clear the Copy Source option.

    Adapt It leaves the Phrase box empty when it encounters an unadapted source text location.

 Note

  • To restore normal process, follow the above steps and click the Copy Source option.
