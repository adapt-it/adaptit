View menu

The view menu contains two sections:

.

  • Tool bar management – this section controls which of the tool and information bars are seen with the main window.

    • The Toolbar option determines if the Toolbar is visible or not.

      Do not hide the Toolbar (Command bar) since some Adapt It commands are only available in the Toolbar.

    • The Status Bar option determines if the Status Bar at the bottom of the Main window is visible or not.

      The Status Bar tells you what project, book folder (if appropriate) and document is currently in use.

    • The Compose Bar option determines if the Compose Bar is visible. (The default is that the Compose Bar is not visible.)

  • Miscellaneous controls – this section manages several functions:

    • The Copy Source option tells Adapt It to copy the word from the source language into the Phrase box when it finds an empty location. If you clear the check mark, Adapt It leaves the Phrase box empty when it encounters an unadapted source text location.

    • Wrap at Standard Format Markers forces a new line of text in the main window when Adapt It encounters a Standard format marker for a chapter or paragraph or other SFM which force a wrap to a new line. (Not all markers force a wrap.).

    • Units of Measurement controls whether Adapt It uses inches or millimeters as the unit of measure for printing documents.

      When you click this option, Adapt It opens a dialog box that allows you to change from inches to millimeters or vice versa. Changing of the units of measurements affects only printing and does not change the appearance of the Main window.
