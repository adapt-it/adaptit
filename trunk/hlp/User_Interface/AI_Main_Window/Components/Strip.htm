Strip

A strip is the horizontal arrangement of words and phrases used by Adapt It. At a minimum, a strip consists of a line of source text and a line of target text that runs across the Adapt It window. Each window contains multiple strips. Each strip may also contain navigation text and glossing text lines. The vertical arrangement of words is called a pile.
