Placeholder

A placeholder is a pile in which the source text cell displays an ellipsis (…). Adapt It creates a placeholder at your request when the target language syntax requires additional adaptation text but the source text at that point does not provide a word or phrase to adapt. You insert the placeholder by clicking the Insert a Placeholder button on the Tool bar. The Phrase box opens a new pile and places the ellipsis in the source text cell. You can then type the needed adaptation text into the Phrase box.

Adapt It also uses a placeholder as part of laying out a retranslation. The application does this automatically.

 Note

  • Do not attempt to type in the ellipsis yourself. Adapt It will not recognize a typed ellipsis as an empty cell marker.
