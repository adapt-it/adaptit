Pile

A pile is a stack of cells arranged vertically in the Main window. Each pile has an entry in the source text cell. The entry in this cell can be a word, phrase, or placeholder. The words or phrases in each pile all line up on the left. (In languages that read from right to left, they would line up on the right.) Each strip in the view window consists of multiple piles. A vertical column of white space marks each boundary between piles.
