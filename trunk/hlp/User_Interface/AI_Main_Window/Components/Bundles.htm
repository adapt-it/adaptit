Bundles

A bundle is the portion of source text in a document Adapt It makes available in the Main window at any one time. (You may need to use the scroll bars to see the entire bundle). Adapt It limits the number of words (or phrases) to keep the application responsive to your actions. The entire source text file is available but you must change bundles to view other portions of the file. The default setting is 300 source text words available in the Main window.

When Adapt It reaches the end of a bundle, it automatically opens a new bundle which includes a new portion of the text. The new bundle includes a portion of the previous bundle so that you can keep the context of your work. In most cases, you will not notice when the bundle changes.

You can directly change the size of a bundle and move to another bundle when you want.

  • Changing the Bundle Size

  • Moving to a New Bundle

 Note

  • When moving between bundles, Adapt It helps to maintain context for you by including words from the previous bundle in the window.

    • When moving down to a bundle, Adapt It includes 50 words of the previous bundle.

    • When moving up to a bundle, Adapt It includes the first 30 words of the following bundle.

  • You cannot use the Scroll bar, Page up or Page Down to move from one bundle to another. These tools operate only within the current bundle. You must use the Move Down One Step or Move Up One Step icons to change bundles. See Moving to a new bundle.
