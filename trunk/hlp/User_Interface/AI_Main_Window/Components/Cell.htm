Cell

A cell is the smallest storage element used by Adapt It. The application uses cells to represent source text, target text, and possibly glosses. A source text cell may contain a word, phrase, or placeholder (…).Target text and gloss cells may contain a word, phrase, or be left empty.

In the Main window, Adapt It organizes cells vertically to form a pile. It also organizes cells horizontally to form a strip. Thus, every cell is a member of both a strip and a pile.
