Moving to a new bundle

Adapt It automatically extends the work area to a new bundle when it reaches the end of the current work area. This action replaces the current bundle with a new set of words from the next part of the document. Any changes made in the previous bundle are preserved in the earlier part of the document.

You can manually move to another bundle do one of the following:

  • Click on the Move Down One Step button in the Tool Bar.

    Adapt It creates a new bundle starting near the end of the previous bundle. There is overlap so that you can maintain the context. The Phrase box is now a short way into the new bundle.

  • Click on the Move Up One Step button in the Tool Bar.

    Adapt It opens a bundle before the first word of the current bundle. The new bundle contains some overlap with the old bundle. The Phrase box is now near the end of the new bundle.

 Note

  • When moving between bundles, Adapt It helps to maintain context for you by including some words from the previous bundle in the window.

    • When moving down to a bundle, Adapt It includes 50 words of the previous bundle. (This setting can be changed in the View tab of the Preferences dialog.)

    • When moving up to a bundle, Adapt It includes the first 30 words of the following bundle. (This setting can be changed in the View tab of the Preferences dialog.)

  • You cannot use the Scroll bar, Page up or Page Down to move from one bundle to another. These tools operate only within the current bundle. You must use the Move Down One Step or Move Up One Step buttons to change bundles.
