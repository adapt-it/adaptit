Changing the bundle size

The default number of source text words is 300, but the user can change the bundle size.

  1. Select Preferences in the Edit menu.

  2. Click the View tab.

  3. Select the Maximum source words (or phrases) displayed at one time (60 to 4000) text box and enter the desired number.

  4. Click OK

 Note

  • The limits for the bundle size are a minimum of 60 words or phrases to a maximum of 4000. Selection of the appropriate number depends on how much of the document you want use for quick access at one time. Be aware that the limitations of your machine may affect Adapt It’s performance. A slow computer may become very sluggish if the bundle size is too large.

  • A small bundle size means that Adapt It automatically changes bundles more often as you work. A small bundle size makes Adapt It more responsive on slow computers, but also means that you have to use the Move Up and Move Down buttons more often.

  • A large bundle size may include more words and phrases than you can see on the computer screen at one time. You can use scrolling to see words or phrases in the bundle but not currently on the screen.

  • You cannot use the Scroll bar, Page up or Page Down to move from one bundle to another. These tools operate only within the current bundle. You must use the Move Down One Step or Move Up One Step buttons to change bundles. See Moving to a new bundle.
