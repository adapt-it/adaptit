Clear the Save To Knowledge Base checkbox

This technique allows you to tell that the source text word or phrase in the active location should never have an adaptation entered into the knowledge base.

  1. If necessary, move the Phrase box to the desired location.

  2. Click the toolbar checkbox Save To Knowledge Base to turn it off.

    Adapt It places the special entry <Not In KB> into the knowledge base. This entry tells Adapt It that this source word will never have a translation stored in the knowledge base.

  3. You now have two options:

    • Leave the word untranslated.

    • Place a translation for the source word at this location in the document.

      The translation is not stored in the knowledge base. Thus, the next time the phrase box encounters the word or phrase Adapt It leaves the target location empty and moves on. Adapt It places an asterisk in the navigation text strip.

 Note

  • The Save To Knowledge Base checkbox stays off only for the current active location. When Adapt It moves to the next location, the checkbox is restored.

  • To undo this action, move the Phrase box to the appropriate location and click the Save To Knowledge Base checkbox. Adapt It allows entry of adaptations into the knowledge base for the source text word or phrase at this location.

  • There are other ways to leave target text empty. See Adapt It types of Nothing.
