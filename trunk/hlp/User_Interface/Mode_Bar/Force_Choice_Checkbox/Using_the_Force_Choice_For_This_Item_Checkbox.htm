Using the Force Choice For This Item checkbox

When beginning a project there are times that you know certain words will require a number of adaptations. You will not want Adapt It to assume that the first translation for a word is the one to be used throughout the document. Thus, you want to override automatic insertion for that word.

On the first occurrence of the word in the Phrase Box:

  1. Click the Force Choice For This Item checkbox so that the checkmark appears.

  2. Type in the first adaptation for the word and Enter.

    Adapt It stores the adaptation into the Knowledge Base, sets a flag which tells Adapt It that the checkbox is turned on fort this location, and moves the Phrase box to the next location.

    When Adapt It again encounters the word, it checks the flag and so instead of inserting the translation and moving on, it opens the Choose Translation dialog, allowing you to accept a previous translation or add a new one.

 Tip

  • If you forget to set the Force Choice For This Item checkbox and find that Adapt It has automatically inserted an incorrect adaptation, you can correct the adaptation by placing the Phrase box on the incorrect target text and do one of the following:

    • Click Show The Choose Translation Dialog button on the Tool bar.

    • Press F8

    • Press Ctrl+L

Note

  • Adapt It maintains the Force Choice settings by each word or phrase it the Knowledge Base. You can change the settings in the Edit or Examine the Knowledge Base dialog.

  • Once a knowledge base entry has two or more adaptations, the Force Choice checkbox is essentially ignored because Adapt It must use the Choose Translation dialog anyway.
