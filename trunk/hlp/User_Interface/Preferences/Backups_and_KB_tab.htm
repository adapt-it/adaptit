Backups and KB tab

The Backups and KB tab of the Preferences dialog box controls how backups of the document and knowledge bases are managed. Be sure that you understand that the first checkbox controls knowledge base backups and the second checkbox controls document backups.

…

Disable automatic knowledge base backups

Normally, Adapt It makes a backup copy the knowledge bases every time you open a project. To keep from making these backups select this check box. If you do check the box, you will not have a backup copy of your knowledge base.

Automatically produce a backup copy when closing the document

Adapt It does not normally produce backup copies of documents. If you wish to have Adapt It make a backup copy, select this checkbox.

Source and Target Language Names Textboxes

These text boxes contain the names of the source and target languages used to name the project. Under extremely rare conditions, these names could become altered. In such a case, you can restore the proper names by typing them into the boxes. You cannot use the textboxes to change the name of a project.

 Note

  • Adapt It backup files have a filename using the name of the project or document and a filename extension of BAK.XML.

  • The two backup checkboxes work in opposite ways. Checking the first box stops knowledge base backups. Checking the second box starts document backups.

  • In glossing mode, Adapt It backups the glossing knowledge base instead of the adaptation knowledge base.
