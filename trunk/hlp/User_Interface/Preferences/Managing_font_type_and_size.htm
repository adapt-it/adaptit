Managing font type and size

To change the font type or size of text, do the following steps:

  1. From the Edit menu, select Preferences.

    Adapt It opens the Preferences dialog box. The Fonts tab should be visible.

  2. Click the Change command button for the desired language category--Source, Target or Navigation.

    Adapt It opens the Font dialog box.

    .

  3. Do one or more of the following:

    • Select a new font in the Font list box. (Use scroll bars as necessary.)

    • Select a new style in the Font style list box.

    • Select a new size in the Size list box. (Use scroll bar as necessary.)

    • Select a new color in the Color drop-down list box.

    The Sample text box shows the results of your selections.

  4. When satisfied, click OK. (Click Cancel to ignore any changes.)

    Adapt It returns to the Fonts tab or dialog box.

  5. Do one of the following:

    • Click OK to finish the font changes.

      Adapt It applies the changes to the document.

    • Return to step 2 and change another type of text.

 Note

  • This dialog box allows you to select from sixteen text colors. For more options, see Managing font color.

  • You will probably not need to use the Strikeout or Underline options in the Effects frame.

  • In the Script dropdown box, Western script should be used unless you are working with one of the identified languages.
