Managing Fonts

You can manage fonts in two locations within Adapt It:

  • When you start a new project, Adapt It opens the to set up your beginning font choices.

  • To change font choices after you have created a project select the Preferences option on the Edit menu.

    Adapt It opens the Fonts tab of the Preferences dialog box.

The accompanying illustration shows the Fonts tab as it appears in Adapt It Unicode. (See Preferences dialog for the Adapt It Regular version.) The organization of the Fonts dialog box is exactly the same. It just does not allow access to other tabs.

The tab contains eight command buttons. Three of these buttons control the font type and the size of the text. The remaining five command buttons control the color of the text. See the following for more information:

  • Managing font type and size

  • Manage font color

Adapt It Unicode can display languages read right-to-left. Select the appropriate option boxes to specify use this format. The above example shows the option boxes as unselected so that all of the languages display in left-to-right format. The option boxes do not appear in Adapt It Regular since this version can only display left-to-right.
