Edit Punctuation Correspondences

Punctuation conventions differ among languages. Adapt It provides a method to establish the correspondence between the punctuation marks in one language and another. To edit these correspondences, do the following steps:

  1. Open the Edit menu.

  2. Click the Preferences option.

    Adapt It enters the Preferences dialog.

  3. Click on the Punctuation tab.

    Adapt It opens the Punctuation tab window. (You need to use the right scroll bar to see all the elements of the window.)

  4. Make the desired changes.

    To make the changes see Managing punctuation.

  5. Click OK.

    Adapt It reparses the document and reopens it. You may occasionally need to make corrections manually.
