Managing preferences

To manage Adapt It preferences, do the following steps:

  1. Select Preferences in the Edit menu.

    Adapt It displays the Preferences dialog box.

    There are eight tabs across the top. Each tab deals with a different topic.

  2. Click the tab for the topic on which you want to work. (The Font tab is the default.)

    Adapt It opens a new page for the selected tab. See the following for more information about each tab:

    • Fonts

    • Backups and KB

    • View tab

    • Auto-Saving tab

    • Punctuation

    • Case

    • USFM

    • Filtering

 Note

  • To set Preferences using the Edit menu, a document from the current project must be open.
