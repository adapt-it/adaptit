Auto-Saving tab

The Auto-Saving tab controls how Adapt It manages the automatic saving of documents and knowledge bases. You should not change these settings unless you have good reason to do so and you understand the implications of making the changes.

Turn OFF periodic saving of the document and knowledge base check box

This check box should be clear. If it is not clear, Adapt It does not automatically save the document or the knowledge base. As a result, if the computer should stop working while you are working on a document, all your changes made in the current session are lost.

Document Auto-Save Criteria frame

This frame contains two option buttons with spin boxes. They give the choice of how Adapt It saves the document and how often. One controls the auto-save operation on the basis of time; the other controls it on the basis of activity. You cannot use both at the same time.

  • Save the document after every NN minutes.

    This option tells Adapt It to save the document to the hard disk after every NN minutes. The default number of minutes is 10.

  • Save the document after every NNN moves of the Phrase box.

    This option tells Adapt It to save the document to the hard disk after you have moved the Phrase box NNN or more times. The default number of moves is 100.

Save the knowledge base at least every NN minutes spin box

This spin box controls how often Adapt It automatically saves the knowledge base if it has not been saved along with the document. The default number of minutes is 15. This option makes sure that the knowledge base is saved regularly even if the document has not been saved.

 Note

  • Adapt It saves the knowledge base automatically every time the document is saved.

  • The Auto-Save spin boxes are grayed out if the Turn OFF periodic saving of the document and knowledge base is checked.
