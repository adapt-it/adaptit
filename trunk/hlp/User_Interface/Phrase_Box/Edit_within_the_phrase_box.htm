Edit within the phrase box

To edit words in the Phrase box, do one or more of the following choices:

  • To select a word in the Phrase box, double-click the word.

  • To select a word or phrase, click at the start of the item and drag the mouse to the end of the item.

  • To select a phrase,

    1. Double-click the first word of the phrase.

    2. Press and hold Shift.

    3. Double-click the last word of the phrase.

  • To delete text to the right of the insertion point, press Delete (or Del) one or more times.

  • To delete text to the left of the insertion point, press Backspace one or more times.

After making the required changes, press Enter to update the knowledge base entry. Adapt It moves the Phrase box to the next empty location.

 Tip

  • When Adapt It moves the Phrase box to a new location, it selects all of the target text for that location. You can replace all of the text by just starting to type a new adaptation. The first character replaces the selected text.

  • Use the Backspace or Delete key, only when you to make minor corrections in the text.
