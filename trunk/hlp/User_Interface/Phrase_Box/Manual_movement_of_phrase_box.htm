Manual movement of the Phrase box

If you notice a misspelled translation in text you have already adapted, you can move the Phrase box to that location for editing.

To move the Phrase box manually, do the following steps:

  1. Locate the source text where you wish to place the Phrase box.

  2. Click in the area immediately below the source text.

  3. Adapt It moves the Phrase box to the new location.

 Note

  • The Phrase box can be moved manually anywhere within the current document except for retranslated phrases.

  • If the selected location is empty (has no previous adaptation), Adapt It takes these steps in the process of moving the Phrase box. After you click on the target text area and before the Phrase box is moved, Adapt It does a lookup of the source text in the knowledge base. As the Phrase box is placed its contents are determined as follows:

    • If the area has no previous adaptation stored for source text word or phrase, Adapt It copies the source text into the Phrase box. (This behavior can be changed. See: Prevent source text from being copied to the phrase box.)

    • If there is a single previous adaptation, Adapt It moves it into the Phrase box.

    • If there are multiple adaptations, Adapt It opens the Choose Translation dialog box. See Using the Choose Translation dialog.

  • If the selected location already has an adaptation, Adapt It simple moves the Phrase box and selects the current target text. It does not have to make a knowledge base lookup.

  • Once you press Enter or move the Phrase box, any changes are stored into the knowledge base.
