Type into the phrase box

The Phrase box is where you enter a translation of the source text.

To enter a new adaptation, do the following steps:

  1. If the text inside the Phrase box is all highlighted, jump to step 3. Other wise, Click inside it.

  2. If the word in the Phrase box is not highlighted, select the text. See Edit within the Phrase box for more information.

  3. Type the new adaptation

    The first character of the new adaptation replaces the original word or phrase then the box accepts the rest of your typing.

  4. Make corrections to your entry necessary.

  5. Do one of the following:

    • Press Enter or Tab to move to the next empty location.

      The Phrase box moves to the next empty location.

    • Press Shift+Tab or Shift+Enter to move backwards one location even if it is not empty.

    Adapt It stores the new adaptation into the knowledge base.

 Note

  • If the content of the Phrase box is almost correct, you may prefer instead to edit the text. Click once right after the part you wish to edit, press Backspace as often as you need to erase letters to the left, and then type what you need. (To erase letter to the right, press Del or Delete.)

  • To remove the highlighting from the phrase box, press the right-arrow key once. No text is now selected and the cursor appears at the right edge of the Phrase box. This action allows you to add more words to the end of the text.
