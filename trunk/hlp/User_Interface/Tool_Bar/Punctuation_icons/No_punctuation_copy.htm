No punctuation copy

Adapt It automatically copies punctuation from the source text to the target text. In most cases this is the desired action. However, sometimes you may wish to tell Adapt It not to copy the punctuation. To do this, do the following steps:

  1. Position the Phrase box at the position where you want to make an adaptation.

  2. On the Tools bar, click the No Punctuation Copy icon.

    Adapt It changes the icon to the Enable Punctuation Copy icon.

  3. Enter the target text for the adaptation

  4. Press Enter.

    Adapt It moves the Phrase box to next open location and restores the icon to the original state. The target text does not have a copy of the source text punctuation.

 Note

  • If the source text uses punctuation symbols with no equivalent in the target language, it is best to enter the source punctuation into the Punctuation Correspondence table but leave the target language punctuation cell empty. See Punctuation management rules for more information.

  • If the target language sometimes uses a symbol for a source punctuation symbol, but sometimes does not use one; Adapt It does not have an automated way of determining whether a symbol should be copied or not. In such cases, you need to make a judgment as to whether it is best to define a target language symbol or to leave the cell empty. If you choose to leave the target cell empty, be sure to make another entry in the table with an empty source language cell and defining the target language symbol. See Punctuation management rules for more information.

    • If you choose to define a target language symbol, you can use the above procedure to prevent Adapt It from copying the symbol.

    • If you choose not to define a target language symbol, you can manually enter the necessary symbol as needed.

    • Using either approach you can manually edit the document in reviewing mode after you have made all of the adaptations.
