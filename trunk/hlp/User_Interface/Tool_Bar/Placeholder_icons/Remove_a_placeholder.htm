Remove a placeholder

To remove a placeholder, do the following steps:

  1. Select the source text placeholder mark (…), or click the target text below the placeholder to make the Phrase box move there.

  2. Do one of the following:

    • On the Tool bar, click Remove A Placeholder

    • Enter Ctrl+D

Adapt It removes the placeholder and the associated text. It positions the Phrase box at the position which followed the placeholder. If that location is a retranslation, the Phrase box moves the first position after the retranslation.

 Caution

When a placeholder is removed, Adapt It deletes the target text from the document. If you reinsert a placeholder, you will need to retype the text.
