Toggle number of target text lines

Adapt It maintains two versions of the target text. One version includes the punctuation and one does not. You can view both versions by using the Toggle 1 or 2 target lines icon. To change the number of lines, do the following:

  1. On the Tool bar, click Toggle 1 or 2 target lines icon.

    Adapt It changes the current number of target lines displayed.

    • If the current number of target lines is one, click the icon to display both lines.

    • If the current number of target lines is two, click the icon to display only one line.

 Note

  • When both target lines are displayed, the target line with punctuation is the bottom line of the strip.

  • When only one target line is displayed, the Show/Hide Punctuation icon controls whether or not punctuation is shown. See Show or hide punctuation.
