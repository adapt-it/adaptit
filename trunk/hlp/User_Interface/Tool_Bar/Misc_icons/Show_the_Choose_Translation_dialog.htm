Show the Choose Translation dialog

You can open the Choose Translation dialog box using the following steps:

  1. Move the Phrase box to the desired location.

  2. Open the Choose Translation dialog box using one of the following options:

    • Click the Show the Choose Translation Dialog button on the Tools bar.

    • Enter F8.

    • Enter Ctrl+L.

    Adapt opens the Choose Translation dialog box. See Using the Choose Translation dialog for information on using this tool.
