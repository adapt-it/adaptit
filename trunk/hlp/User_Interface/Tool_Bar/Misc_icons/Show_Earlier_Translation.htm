Show Earlier Translation

Click the Show Earlier Translation button to cause the View Translations or Glosses Elsewhere in the Document dialog to appear. See Using the View Translations or Glosses Elsewhere dialog for more detailed information.

The dialog permits you to examine a translation that you did at a different location in the document. This location is specified by a chapter and verse reference. This location can be any point in the document away from where the phrase box is currently located. You can use the dialog to:

  • Show the source and translation or gloss of the verse

  • Expand the context of the verse by showing the previous and next verses

  • Close the window and move the Phrase box to the first word or phrase of the verse.

You can hide the dialog by clicking anywhere in the Main window, or you can close it by clicking the Close button.

 Note

  • Glossing must be on in order to see glosses in the dialog. When glossing is on, only the gloss appears and not the translation.
