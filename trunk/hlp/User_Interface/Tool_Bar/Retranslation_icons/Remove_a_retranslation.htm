Remove a retranslation

To remove a retranslation, do the following steps:

  1. Select one or more of the source text words above the retranslation.

    Adapt It makes the Remove Retranslation button in the Tools bar available.

  2. Click Remove Retranslation

    Adapt It removes the retranslation and begins the adaptation process for the source text words.

  3. Respond to any adaptation requests as you would in any other situation.

 Note

  • Adapt It places the retranslation text into the text box of the Compose bar. The text remains in the text box until something else replaces it or you remove it. See Compose bar for more information about this tool.
