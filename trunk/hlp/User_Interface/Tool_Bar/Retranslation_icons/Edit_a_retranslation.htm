Edit a retranslation

To edit a retranslation do the following:

  1. Select one or more of the source text words above the retranslation.

    Adapt It makes the Edit Retranslation button in the Tools bar available.

  2. Do one of the following:

    • Click Edit Retranslation

    • Enter Crtl+E.

    Adapt It opens the Type Retranslation Text dialog box. See Using the Type Retranslation Text dialog on how to use this dialog.

  3. Click OK to insert the new retranslation into the document.
