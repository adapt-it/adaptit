Retranslate selected source text

With Adapt It, the term retranslation means that you enter your own translation of a passage rather than accepting the adaptation produced by Adapt It. Generally, you enter retranslations when the syntax or exegesis of the adaptation is incorrect or misleading. You also need to retranslate if you need to group more than ten source words to make a phrase.

To retranslate the source text, do the following steps:

  1. Select the source text to be retranslated.

  2. Click the New button on the Tools menu.

    Adapt It displays the Type Retranslation Text dialog.

    See Using the Type Retranslation Text Dialog.

  3. Once the dialog appears you can use one text box and four command buttons:

  • Type the new translation here

    This text box allows you to edit or retranslate the translation text that Adapt It has copied here.

  • Copy to Clipboard

    This command button copies the selected text from the Type the new translation here text box into the Clipboard. You can then paste that text to some other location. See Clipboard.

  • Show Target Context

    This command button changes the contents of the context text boxes to show them in the target language rather than the source language. To return to the source language, simply click the button again. The button now reads Show Source Context.

  • OK

    This command button applies the changes and closes the dialog box.

  • Cancel

    This command button ignores any changes and closes the dialog box.
