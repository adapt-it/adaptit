Add a note

To add a note, do the following steps:

  1. If the Phrase box is where you want to add a note, skip down to step 2. Otherwise, click the source or target text at the word or phrase where you want to add the note.

  2. Click the Open a Note Dialog icon on the Tool bar.

    The Note dialog appears.

  3. Type the note in the dialog's edit box. If the note is longer than the note dialog can display at one time, use the dialog's scroll bar to view the entire note. See Using the Note dialog for more information.

  4. Click on the dialog's OK button to save the note.

    The Note dialog closes and Adapt It adds the Note and Filtered icons above and preceding the source text where the note is stored.

    The yellow colored icon on the left represents the actual note that you added. The green wedge next to it represents all filtered information stored at that location (including the note).
