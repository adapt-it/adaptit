Using the Print Setup dialog

The Print Setup dialog consists of three sections and four command buttons.

They include:

Printer

You can choose from a list of printers that have been defined to your computer. To choose a printer:

  1. Click on the down arrow in the Name: drop-down box.

  2. Click the name of the printer you want to use.

The Properties command button allows you to change the options for any listed printer. Each type of printer has its own set of properties and we cannot show them here.

Paper

In this section, you can select the type of paper the printer should expect and the tray to use. In most cases you will use A4 or Letter format.

To select the paper size

  1. Click on the down arrow in the Size drop-down box.

  2. Click the paper size you want to use.

The Source combo box allows you to select a paper tray if the chosen printer supports multiple trays.

Orientation

In the Orientation section, you determine if the page should be printed in Portrait orientation (page is arranged vertically) or Landscape orientation (page is arranged horizontally).

Click on the desired option button. Normally the Portrait option button is the default.

Help

The Help command button opens this help file.

Network

The Network command button allows you access a printer on a network and does not apply to a standalone computer. If you are on a network, ask your network administrator for advice.

OK

The OK command button accepts any changes you have made and returns to the Adapt It main window.

Cancel

The Cancel button restores any changes to the previous settings and returns to the Adapt It main window.
