Using the Print dialog box

This dialog box allows you to control how the document is printed. There are three sections in this dialog box:

  • Printer

  • Print range

  • Number of Copies

  • Once you have determined your choices, do one of the following:

  • Click OK to accept the choices and close the dialog box.

  • Click Cancel to ignore the choices and close the dialog box.

 Note

  • Adapt It normally prints a footer line at the bottom of the page. This footer contains:

    • Project name

    • Document name

    • Page number

    • Date and time printed.

    To keep this line from printing, click in the Suppress printing of the footer check box

  • When you use the Chapter/Verse Range option, depending on the verses you choose, you may or may not print the section header information.

    • If your selection starts at the first verse of a section you do not want to see section header information printed, check the Suppress a preceding section heading checkbox.

      Adapt It prints only the selected verses and navigation information. It does not present the header information.

    • If your selection ends just before a new selection and you want to see the section header information of the next selection printed, check the Include a following section heading checkbox.

      Adapt It prints the selected section of verses but also prints the section heading information for the immediately following section.
