Number of Copies

In this section you can tell Adapt It to print more than one copy.

To print multiple copies, select or type the number of copies in the Number of copies list box.

If you select more than one copy, Adapt It enables the Collate check box.

  • If you check the box, Adapt It prints copies in sets.

  • If you do not check the box, Adapt It prints the number of copies grouped by page. For example, if you choose 2 copies, Adapt It prints 2 copies of page one, 2 copies of page two, etc.
