Using the View Translations or Glosses Elsewhere dialog

You can use this dialog to find and view translations at other locations in the document. To use the dialog, you need to know the chapter and verse of that location. To use this dialog, do the following steps:

  1. On the Tools bar, click the Show Earlier Translation icon.

    Adapt It displays the View Translations or Glosses Elsewhere in the Document dialog box. The two large text boxes are empty until you choose a chapter and verse.

  2. Set the numbers of the desired chapter and verse in the Chapter and Verse list boxes. The default values are “1” and “1.”

  3. Click the Get Chapter:Verse Text command button.

    Adapt It places text into the two large text boxes. The source text goes into the upper box. The translation or gloss text appears in the lower box.

  4. At this point you can do one of the following:

    • Click the Show More command button to see more of the context.

      Adapt It expands the number of verses in the textbox by adding a verse before and a verse after the current selection.

      You can use this command button multiple times to expand the context.

    • Click the Show Less command button to see less of the context. This button is available only when the text boxes contain more than one verse.

      Adapt It reduces the number of verses in the textbox by removing the first and last verses of the current selection.

    • Click the Close and Jump Here command button to move the Phrase box to the first word or phrase of the selected verse. This action also closes the dialog box.

    • Close the dialog by clicking the Close command button or by clicking anywhere in the Main window.
