Move a document dialog

If you have not been using Book folder mode and then turn it on by using the Storing Documents In Book Folder option of the Advanced menu, you must move documents from the Adaptations folder to the appropriate book folder. To accomplish this, do the following steps:

  1. Open a project.

    Adapt It opens the Document dialog.

  2. Determine which book folder is currently open.

    If you wish to change the folder, click the Change Folder… command button to open the Choose a book folder dialog and select the folder you want to use.

  3. In the Document dialog do one of the following

    • Open a document by selecting one of current documents and clicking Finish.

    • Open the Main window without a document by clicking Cancel.

      Adapt It remembers which folder is selected.

  4. Select the Move Document option of the Tools menu.

    Adapt It opens the Move A Document dialog.

  5. Notice the list box to the left and the option buttons on the top right.

    The list box contains the names of documents that you can move.

    The option buttons determine which way you can move the documents:

    • From the Adaptation folder to the current book folder (default)

    • From the current book folder to the Adaptations folder

    As you change the select option button, the caption above the list box on the left changes and contents of the list box change to reflect a list of the document in the selected folder.

  6. To see what documents are already in the folder that is to receive the documents, click the View The Other Folder's Documents… command button.

    Adapt It opens the Documents in Other Folder dialog.

    The list box reflects of the documents in the receiving folder. This dialog is for information purposes only. Click Close to close the dialog.

  7. Select the document in the list box that you want to move.

  8. If you wish to rename the document click the Rename Document… command button.

    Adapt It opens a dialog box that allows you to enter a new name for the document. Type in the new name and click OK. Otherwise, click Cancel to ignore the changes.

  9. When you are satisfied, click Move Now… to move the document.

    Adapt It moves the document into the new folder.

  10. Do one of the following:

    • Repeat steps 7 through 9 for each document you wish to move.

    • Click Close to return to the Main window.

 Note

  • The steps are the same for moving the document in either direction. You control the direction by selecting one of these options:

    • From the Adaptations folder to the current book folder checkbox (Default)

    • From the current book folder to the Adaptations folder checkbox

  • You can move only one document at a time.

    In the case where you have Split a document you may have many small documents. If you join these documents back together, the task is easier and you have fewer documents to manage.

  • See Choose a book folder dialog for information on how to change folders.

  • If you move the current document, Adapt It removes it from the Main window.
