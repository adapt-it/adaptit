Print Setup Dialog Box

This dialog box allows you to specify the printer to be used for printing documents. This dialog box appears when you select the Print Setup option in the File menu. (You see a different dialog box when you select the Print option.)

There are three aspects of the printer that can be controlled:

  • Printer

    The printer can be specified by choosing an available printer in the Name combo box. Click the Down arrow icon to see more printers.

    The Properties command button presents a dialog window allowing you to control settings for each printer. Since there are many possible printers, we do not show this dialog box.

  • Paper type

    You can choose a standard paper size by using the Size combo box. The specific options vary based on the printer type chosen above.

    The Source combo box allows you to select paper trays if the chosen printer supports multiple trays.

  • Print orientation

    You can select Portrait orientation (where the page is arranged vertically) or Landscape orientation (where the page is arranged horizontally) for how the page will be printed. Click the appropriate option button -- Portrait is the default. (The Page icon changes when you make a choice.)

There are also four general command buttons:

  • Help – Opens these help files.

  • Network – Allows you to find a printer on a network. Works only on networked printers.

  • OK – Selects the desired printer and printing options. Closes the dialog window.

  • Cancel – Closes the dialog window without making any changes.
