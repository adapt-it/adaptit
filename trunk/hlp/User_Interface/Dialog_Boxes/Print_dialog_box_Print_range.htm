Print range

In this section you can control what will be printed. There are four option buttons and two check boxes.

  • All – this option is normally the default and will print the entire document.

  • Selection – in most cases this option is grayed out, meaning it cannot be used. However, if you have selected a passage in the document the Selection button is made active and becomes the default. In this case, Adapt It prints only the selected passage.

  • Pages – with this option, you can choose the pages to print. Adapt It gives you the starting and ending pages of the document in the from and to textboxes. You can change either or both values. If you want to print only one page, set both boxes to the same value. That is, if you put “2” into both boxes, only the second page prints. Do not try to print pages in reverse order.

  • Chapter/Verse Range – this option allows you to print a specific range of verses. Adapt It enables four text boxes.

    • From: chapter: -- Starting chapter of the desired range.

    • From: verse -- The starting verse of the starting chapter.

    • To: chapter -- The last chapter of the desired range.

    • To: verse -- The last verse of the desired range.

    Type the desired values in the boxes. Do not try to print verses in reverse order.

  • Suppress a preceding section heading – if the document includes section headings (see Standard format markers) you can prevent any that precede the passage from printing by checking this check box. This selection can be used with any of the option buttons and any of the other check boxes.

  • Suppress a following section heading -- if the document includes section headings (see Standard format markers) you can prevent any that comes after the passage from printing by checking this check box. This selection can be used with any of the option buttons and any of the other check boxes.
