Edit or examine the knowledge base dialog

The knowledge base editor uses a dialog box for its user interface (UI). This dialog box has several capabilities and the user must understand the interface to take advantage of its resources.

  • Source Phrase tabs

    These tabs are across the top of the dialog and allow you to choose words or phrases by the number of words in the phrase.

  • Source Phrase list box

    This list box is the large box on the left side of the dialog. It contains a list of all source words or phrases as classified by the selected tab.

  • Existing Translation list box

    This list box is a list box on the top right of the dialog. It contains a list of all the translations for the word or phrase selected in the Source Phrase list box.

  • Edit or Add a Translations text box

    This text box is in the middle right side of the dialog. It contains a copy of the selected word or phrase in the Existing Translation list box. You can use this text box to edit the selected translation or to add a new translation.

  • Edit and Arrange command buttons

    These command buttons are located immediately below the Edit or Add a Translation text box. You can use them to:

    • Update the selected translation with any changes in the Edit or Add a Translation text box.

    • Add an entirely new translation to the list.

    • Remove the selected translation from the list.

    • Move the selected translation up or down in the list.

    • Add a <no adaptation> entry to the list.

  • Toggle the Setting button

    This button changes the Force Choice For This Item checkbox status to On or Off.

  • Type Key To Be Found text box

    This text box is located in the bottom center of the dialog. If type the initial letters of a source word or phrase, Adapt It finds the matching word or phrase and selects it in the Source Phrase list box.
