Using the Inconsistency Found Dialog Box

The Inconsistency Found dialog box has several areas to notice. The colors have been added for reference purposes.

  • The upper left corner (highlighted in yellow) contains the word or phrase from the source language. There is also a listing of the location for the entry, which in this case is chapter 25, verse 1.

  • The upper right corner (highlighted in blue) contains the word or phrase in the target language as it is translated in the current document. This adaptation is not currently in the knowledge base.

  • The list box on the left side shows any current translations for the words or phrases that are in the knowledge base associated with the source word shown at the top left, which is toktok.

  • In the center of the right hand side are three option buttons and a text box (highlighted in green) used to tell Adapt It how to deal with the inconsistency.

    • Select from the list on the left, to replace what is above. To use this option, select one of the adaptations in the list box on the left.

    • Accept what is above. This option enters the adaptation at the top into the knowledge base (in the blue area).

    • Type a new one. This option allows you to enter a new translation to be placed in the document and added to the knowledge base. Type the new translation into the text box to the right of this option.

  • The bottom right corner (highlighted in magenta) contains four command buttons and a check box.

    • Accept <no adaptation> for this item. This command button enters the <no adaptation> entry into the database and clears the contents of the Phrase box.

      Click OK to have Adapt It make these changes and to resume consistency checking at the next location in the document.

    • Ignore it, I will fix it later. This command button tells Adapt It to do nothing and move on to the next location.

    • Auto-fix the entry the same way. This check box tells Adapt It to use the selected options (in the green area) for all future occurrences of this particular discrepancy. (You must click one of the command buttons to continue.)

    • OK. This command button causes Adapt It to process the chosen option and resume consistency checking from the next location in the document.

    • Cancel. This command button causes Adapt It to stop the consistency checking process; no changes are made for the current selection.
