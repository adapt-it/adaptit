Remove Unwanted Files from the List

Under several conditions, Adapt It allows you to select the files you want to include in an operation. It does so by opening the Remove Unwanted files from the List dialog box.

Do one or more of the following options.

  • Select the name of any file in the left hand box that you do not want to use and click the Red X.

    Adapt It moves the filename to the right hand box. Repeat this process until all unnecessary files have been moved.

  • Highlight any files in the right hand box that you do want to use and click the Green check mark .

    Adapt It moves the file to the left hand box. Repeat for any other files in the right hand box you want to use.

  • Click OK to process all files in the left hand box.

  • Click Cancel to quit the process.
