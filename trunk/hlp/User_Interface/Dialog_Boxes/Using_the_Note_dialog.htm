Using the Note dialog

The Note dialog allows you to create or edit Adapt It notes and to navigate among the notes.

Text Box

The large text box contains the contents of a single note and you can use this box like a text editor to modify the note. Thus, you can add or delete text as well as editing the current text.

Search by Key Word

At the bottom of the Note dialog is a small text box and the Find Next command button. You can use these tools to find other notes in the document by typing in some search text, which can be a specific word, part of a word, or a phrase. Type the search text into the small text box and click Find Next. Adapt It moves ahead to the next note and looks for the search text, it stops at the first note that contains the search text. If Adapt It does not find the search text in any note it displays the following message box:

Notice that the message says “Searching forward did not find a note…”. This means that Adapt It only searches from the current note to the end of the document. To search the entire document, you need to move to the first note.

Delete

The Delete command button at the bottom of the dialog removes the current note from the document. Use this button only if you are sure that you do not want this note any more. Adapt It does not give you a warning and cannot restore the note except for you to type it in again.

Navigation Buttons

The dialog contains four navigation buttons:

  • <<First

    This button opens the dialog for the first note in the document.

  • <Prev

    This button opens the dialog for the note immediately before the current note.

  • Next>

    This button opens the dialog for the note immediately after the current note.

  • Last>

    This button opens the dialog for the last note in the document.

Closing the Dialog

You can close the dialog by clicking one of two command buttons:

  • OK

    Adapt It saves the current changes in the large text box and closes the dialog.

  • Cancel

    Adapt It closes the dialog but does not save any changes.

 Note

  • You can move a note forward or backward without opening this dialog. The Edit menu contains commands that can move the note. You can also use Ctrl+3 to move the note toward the end of the document, or Ctrl+2 to move the note toward the front of the document. See Moving notes.
