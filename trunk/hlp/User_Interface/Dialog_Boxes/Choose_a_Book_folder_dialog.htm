Choose a Book folder dialog

The Choose A Book Folder dialog appears when you have selected Choose Folder from either:

  • The Change Folder… option of the File menu

  • A Change Folder command button in one of several dialogs that Adapt It displays.

The Choose A Book Folder dialog appears:

To select a new Book folder, do the following steps:

  1. Select a general category of books by selecting the option button next to that category.

    In the example above, the New Testament books option has been selected.

  2. Click on the down arrow in the combo box on the right.

    A list of all the book folders in the selected category appears.

  3. Select the desired book folder.

    Adapt It closes the list and the name of the selected folder appears in the box.

  4. Do one of the following:

    • Click OK to have the chosen book folder accepted, and then the dialog closes.

    • Click Cancel to close the dialog without changing the folder.
