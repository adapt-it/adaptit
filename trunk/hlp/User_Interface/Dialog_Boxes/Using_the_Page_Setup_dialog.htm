Using the Page Setup dialog

The Page Setup dialog box allows you to control the margins in the output page.

These settings affect both the printed output and the print preview output. This dialog is deliberately limited in its options. You cannot change the paper size nor set the page orientation. See Using the Print Setup dialog to change the paper size or page orientation.

To change the margins for the output do the following:

  1. Type the size of the margin in any of the four text boxes you wish to change.

    The picture of the page and its margins adjust as you make the changes.

  2. Click OK to accept the changes.

    Adapt It makes the changes and closes the dialog.

Command buttons

There are four command buttons:

  • Help – Opens this help file.

  • OK – Saves the changes and closes the dialog

  • Cancel – Ignores the changes and closes the dialog.

  • Printer… – This button does not have any effect in this dialog.
