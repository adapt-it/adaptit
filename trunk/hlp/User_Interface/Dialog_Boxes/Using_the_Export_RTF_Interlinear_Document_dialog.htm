Using the Export RTF Interlinear Document dialog

The Export RTF Interlinear Document dialog manages the contents of the exported interlinear document. The output document resembles the organization of the main window so that navigation, source and target text are printed in parallel lines.

There are four sections in the dialog, each controlling some aspect of the exported document. The sections are:

  • Primary Rows to include in exported tables.

    This section lets you determine which language texts to include in the output. You can choose:

    • Navigation Text

      Default option. This includes the chapter and verse information as well as Id and other information.

      Click the check box to keep this text from appearing.

    • Source Text

      Default option. The source language appears in the document.

      Click the check box to keep this text from appearing.

    • Translation (Target) Text

      Default option. The target text appears in the document.

      Click the check box to keep this text from appearing.

    • Gloss Text

      Is enabled only when the See Glosses command in the Advanced menu is turned on. (See Glossing overview to see how to allow to make any typed glosses to become visible.)

      Click the check box to include the Gloss text.

  • Maximum output table width designed for:

    This section allows you to decide if the output page should be aligned in Portrait Orientation (where the page is aligned vertically) or Landscape Orientation (where the page is aligned horizontally). Portrait orientation is the default.

    Click the Landscape Orientation option button to change the print orientation.

  • Output Range

    This section allows you to determine how much of the document should be printed.

    • All

      This option is the default.

    • Chapter/Verse Range

      This option controls what verses are to be printed.

      Click the option button to select.

      Adapt It enables four text boxes.

      From: chapter: -- Starting chapter of the desired range.

      From: verse -- The starting verse of the starting chapter.

      To: chapter -- The last chapter of the desired range.

      To: verse -- The last verse of the desired range.

      Type the desired values in the boxes.

    • Preliminary Material Only

      This option outputs only the material that occurs before the Chapter and Verse text. This can include such things as Preface, Introduction and Outline. (See Standard format markers) for further information on these items.)

    • Final Material Only

      This option outputs only the material that occurs after the Chapter and Verse text. Final material includes such things as the Glossary or Topical Index.

  • Paragraph breaks and centering

    This section controls how paragraphs are handled based on various standard format markers used in the document. There are two checkboxes for controlling the way paragraphs are organized.

    • Start new tables for markers that start new lines…

      When checked, Adapt It starts a new set of lines in the output document when it encounters a SFM that require a new line. If the box is unchecked, Adapt It ignores the new line requirement.

    • Center the tables for markers that use centered justification…

      This check box is available only when the previous checkbox is checked. When checked, Adapt It centers the paragraph with equal spaces on each side. If the box is unchecked, Adapt It ignores the centering requirement.

Once all options are selected: Choose one of the following options:

  • Click OK

    Adapt It opens a Save As dialog box and allows you to save the file. You can change the file name if you wish.

  • Click Cancel

    Adapt It cancels the export process and returns to the Main Window.

  • Click Export Filter/Options

    Adapt It opens the Export Filter/Options dialog. See Export filter options for more information.
