Installing Adapt It

To install Adapt It for the first time do the following steps:

  1. Download the executable files from one of the following sources:

  • http://adaptit.ebt.cx/

  • http://carla.sil.org/Corporate_Release/AdaptIt/

  • You can also use a copy of the executable files from another translator if available

    If you are using one of the Internet sites, be sure to select the version and language that is appropriate for your project.

  1. Run the program.

    You may see a warning message that the source of the file is not known. You can safely ignore this message and run the program.

  2. Adapt It opens an Installing Adapt It dialog.

    This dialog determines where the Adapt It executables are stored. It is best not to change the Destination Directory.

  3. Click Start.

    The installer program creates an Adapt It or Adapt It Unicode subfolder in the computer’s Program Files folder.

    Adapt It displays and option box allowing you to control what you want to see next.

  4. Select one or both of the options. The default is to select both.

    • View Readme File

      Adapt It display a text file in NotePad. This file tells you information about the latest release of Adapt It.

    • Run Installed Application

      The installer launches Adapt It.

 Note

  • The Adapt It subfolder in the Program File folder is created the first time you install Adapt It. If you install a new version of Adapt It, the installer uses the Adapt It subfolder and updates the executable files.

  • No adaptation data files are modified by the installation of a new Adapt It version.
