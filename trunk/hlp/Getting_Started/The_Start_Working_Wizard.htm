The Start Working… wizard

After launching Adapt It, the Start Working… wizard appears. This wizard allows you to open a previously created project or to create an entirely new project. The wizard directs you through the steps needed to create a project, set up fonts and colors for the different kinds of information Adapt It knows about, and then to create and open your first document for doing adaptation work.

You can use the Start Working… wizard at any time by selecting it in the File menu.
