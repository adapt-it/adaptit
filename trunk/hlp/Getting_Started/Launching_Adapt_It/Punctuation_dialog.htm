Punctuation dialog

The punctuation dialog opens as part of the creating a project sequence. The dialog box is where you should specify all of the punctuation symbols which are to be used in both source and target languages. Be sure that symbols that are the same in both languages are the boxes next to each other. See Managing punctuation for more information.

  1. Add, change, or delete any punctuation marks as necessary for use with your source and text languages.

  2. Do one of the following:

    • Click < Back to return to the Fonts dialog.

    • Click Next > to proceed to the Case dialog box.

 Note

  • The Punctuation dialog works in the same way as the Punctuation tab in the Preferences dialog box.
