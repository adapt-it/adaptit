Case dialog

In the project set up process, Adapt It allows you to set upper and lower case equivalences and to enter alphabetic characters unique to your languages.

When Adapt It opens the Case dialog, do the following:

  1. Read the explanation in the dialog.

  2. Add any necessary characters in the one or more of the columns as appropriate for the languages you are using. You do not have to type in the capitalized letters in the target language.

  3. Scroll down as necessary.

  4. Use the buttons at the bottom of the scrolled screen to perform any required operations.

    • Clear Source List (or Target or Gloss) removes all of the characters in the Source (or Target or Gloss) column.

    • Set English Equivalences restores the default English character set into the column above the button.

    • Copy to Target List (or Gloss or Source) copies the character set in the column above the button to the designated column.

    • Use Automatic Capitalization tells Adapt It to follow the capitalization found in the Source language. You do not have to type in the capitalized letters in the target language. This button also tells Adapt It to store the word in the knowledge base only once, instead of one entry for the capitalized word and another for the lower case word.

  5. Click Next to move to the next option.

 Note

  • You can also turn Automatic Capitalization on or off using the Automatic capitalization option in the Tools menu. See Automatic capitalization for more information.

  • Turning Automatic Capitalization on and off from time to time may be done safely. The only impact may be that the knowledge base may contain two entries for a single word -- one in lower case and one that is capitalized. If Automatic Capitalization is on, the capitalized entry is ignored and the lower case entry is used. There is no need to remove the capitalized entries from the knowledge base as their presence is benign, and in fact may be useful if you turn off Automatic Capitalization.
