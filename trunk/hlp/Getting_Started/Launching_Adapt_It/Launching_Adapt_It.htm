Launching Adapt It

To Launch Adapt It do the following:

  1. Start Adapt It using one of these choices:

    • Double-click the Adapt It Icon on the desktop.

    • Click the Start button and select Adapt It from the Programs menu.

    • Adapt It starts and displays a Welcome window.

    • To prevent this window from appearing again, click the Do not show in future check box.

  2. Click OK.

    Adapt It displays the Project dialog box.

  3. Do one of the following:

    • Create a project.

    • Opening a Previously Created Project
