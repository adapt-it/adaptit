USFM dialog

The filtering dialog appears as part of the process of creating a new project. This dialog allows you to set your preferences for the Standard Format Marker Set. The dialog looks like this:

See Set USFM Preferences for information on using this dialog.

After making any desired changes, do one of the following options:

  • Click Next to move to the Filtering dialog.

  • Click Back to return to Case dialog.

  • Click Cancel to close the project and leave the Main window empty.

  • Click Help to show these helps.

 Note

  • Changes involving the Standard Format Marker (SFM) set should be made here in the USFM page before making any Filtering changes in the Filtering dialog.
