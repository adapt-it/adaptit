Fonts dialog

The Fonts dialog appears in the Startup process. This example is the Adapt It Unicode version. See Non-Unicode Fonts dialog for another version.

This dialog contains the same entries as the Fonts tab of the Preferences dialog box except that it has navigation buttons and does not have access to other Preferences tabs. See Managing Fonts to change the default settings.

  1. Do one of the following options:

  • Click Next to move to Punctuation dialog.

  • Click Back to return to Languages dialog.

  • Click Cancel to close the project and leave the Main window empty.

  • Click Help to show this topic.
