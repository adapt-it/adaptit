The Startup Wizard

After launching Adapt It, the Startup Wizard appears. This wizard allows you to open a previously created project or to create an entirely new project. The wizard directs you through the steps needed to create a project , set up fonts and colors for the different kinds of information Adapt It knows about, and then to create and open your first document for doing adaptation work.

You can use the Startup Wizard at any time by selecting the Start working option in the File menu.
