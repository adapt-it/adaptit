Importing overview

Importing is the process that allows an application to use a file or document prepared by another application. Generally, the import process copies the file into a new format. For example, when Adapt It creates a new document, it imports a “txt” file to use as the source text. The txt file would have been prepared in a word processor or another translation application. As Adapt It imports the file, it parses (splits up) the file into a list of structured objects. These objects are used by Adapt It in its processing. The program stores those objects in an “XML” file for use in between adaptation sessions. The import process does not make any change to the original document or file.

Adapt It can import:

  • Source text

  • Knowledge base entries
