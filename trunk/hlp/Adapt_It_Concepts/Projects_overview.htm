Project overview

Adapt It uses projects to organize translation activities. A project is a collection of files necessary for Adapt It to do its processing. These files consist of adaptation documents and a knowledge base for a pair of languages. Of these two languages, one is the source language for the texts being adapted. The other language is the target language for the adaptation. Each adaptation document contains the source text and the portion of target text that has already been adapted.

 Note

  • While only one knowledge base is required for a project, Adapt It also maintains a glossing knowledge base if the project uses glossing.
