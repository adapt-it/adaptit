Knowledge base overview

Knowledge bases are special files used by Adapt It to manage adaptations of source text. These files are empty when you first start a project. However, as you begin to adapt the source text, Adapt It captures each unique word or phrase in the source language text and the corresponding adaptation word(s) you supply in the target or glossing language. It then stores these corresponding words and phrases into the knowledge base. As Adapt It encounters a source word or phrase that has already been entered, the application then places the target adaptation in the knowledge base into the target text. So you don’t have to reenter these adaptations. The more words you adapt, the “smarter” Adapt It becomes.

Adapt It maintains knowledge bases for each project and uses the same knowledge base for all documents in the project. There are two types of knowledge bases:

  • Adaptation knowledge base

  • Glossing knowledge base
