About Adapt It

Adapt It is a computer application that assists language translators in the conversion of text from one language to another. Since Adapt It is designed for use with languages that are closely related, we use the term adaptation (rather than translation) to describe the process. With adaptation, the exegesis of the translation being done in the target language is dependent upon the translation done earlier in the source language.

The heart of Adapt It is its use of knowledge bases. These files capture the adaptations of source text words and phrases. The knowledge base allows Adapt It to automatically apply these adaptations to the same words and phrases as it encounters them later in the text. See Knowledge base overview for more information.

In addition to the basic adaptation process, Adapt It has a number of capabilities that can assist you in creating translations. See What tasks can Adapt It do? for more information.
