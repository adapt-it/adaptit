How adaptation is done

Adaptation is a process of converting text from one language to another closely related language. The technique is particularly effective when the two languages have similar grammatical structures and the words of one language are related to the words of the other. Thus, a translator can create a new translation in the second language by adapting a translation in the first language. We call the first language the source language and the second language the target language.

Adapt It is a computer application that automates a large part of the adaptation process. This program is not required to do adaptation but it can significantly speed up the process. See How Adapt It does adaptation.
