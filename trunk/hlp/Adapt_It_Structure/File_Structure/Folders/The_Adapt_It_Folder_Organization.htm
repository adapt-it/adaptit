The Adapt It folder organization

When the user installs Adapt It, the installation process creates a folder structure to support the adaptation process. This structure works at several levels:

The Adapt It or Adapt It Unicode Work folder

This folder is the highest-level folder and contains all of the subfolders necessary for the adaptation process. Adapt It creates the Work folder as a subfolder of the My Documents folder. The Work folder contains the Project Adaptation folders. It also contains a configuration file that defines the Adapt It user interface preferences.

Project Adaptation folders

Project Adaptation folders are subfolders of the Work folder. Each adaptation folder represents one project. The folder name contains the name of the project, for example Tok Pisin to English Adaptations. Each folder contains an Adaptations subfolder and several files that Adapt It needs to perform the adaptation process. The most important of these files is the knowledge base file.

The Adapt It application maintains these files and the user should not attempt to modify them in anyway other than through Adapt It tools.

Adaptations subfolders

The Adaptations subfolder contains the document files for the project or Book subfolders. Adapt It maintains these files and the user should not attempt to directly read or modify them in anyway other than through the Adapt It application.

Book subfolders

If Book folders mode is on Adapt It uses another level of subfolders. The Adaptations subfolder contains the book subfolders. There are 67 of these subfolders -- one for each book of the Bible and a generic folder called Other Texts. Each of the book subfolders contains documents for a particular book of the Bible.

This arrangement provides several benefits. It keeps the number of documents in a subfolder to a manageable size so that you do not have to create a new project for each new book. In addition, the knowledge base created for one book is instantly available to all of the other books. You do not have to export knowledge base entries to another project each time you start on a new book.

 Warning

  • Do not attempt to start Adapt It by opening a document file directly. Adapt It cannot be started by clicking on a document file. Start Adapt It through the normal application process.
