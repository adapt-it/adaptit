Data formats acceptable to Adapt It

The two data formats acceptable to Adapt It are the American National Standard Institute ANSI standard and Unicode.

ANSI

The ANSI standard is an older data format that has been used with computers for many years. This standard has worked well with Latin-based alphabets because each ANSI character can be one of 255 possibilities. Non-Latin-based alphabets can be much more complex and may require many more possibilities.

Unicode

The Unicode standard is a newer data format which allows 65536 combinations or more and thus can be used for complex non-Latin based fonts. However, Unicode can only be used with Windows 2000 or newer machines since the older computers and operating systems cannot use this format.
