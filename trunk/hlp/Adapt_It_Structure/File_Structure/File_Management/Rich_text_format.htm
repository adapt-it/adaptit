Rich text format (RTF)

Rich text format (RTF) is a file format that many word processing programs understand; it is often used when a document is created in one word processing program but is expected to be edited in another word processing program. Adapt It can export files in RTF so that they can be edited in MS Word. Adapt It uses the same formatting styles as Paratext and the Word Scripture Template for publishing purposes.

 Note

  • While Adapt It can export documents in RTF format, it cannot import them. That is, Adapt It cannot read RTF files. Adapt It can only read “txt”, “adt”, and XML files.
