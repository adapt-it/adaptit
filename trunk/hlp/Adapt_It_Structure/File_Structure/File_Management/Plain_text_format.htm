Plain text format (TXT)

Plain text format is a file format that includes only letters and symbols (such as punctuation) that can be typed from the keyboard. These files usually have a filename extension “txt.” Plain text format is used by the MS Notepad program although most word processors can import or export this file format. Adapt It can read and export TXT files.

 Warning

  • Files with a “doc” or “rtf” filename extensions are not plain text files. Adapt It cannot read files with either of these extensions.
