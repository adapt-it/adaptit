A project is a collection of all the files and documents necessary for Adapt It to carry out its work. Each project deals with two languages: the Source language and the Target language. The input documents which are to be adapted are in the in the source language. The output documents of the process are in the target language. Adapt It stores the projects and its files in a computer folder. This folder is usually named for the two languages used by the project. The project folder contains the knowledge bases and at least one subfolder named Adaptations. The adaptations folder contains the adaptation documents, either in the subfolder or one or more of the Book folders.

Project folder

The project folder is created when you first start the project. (See Create a project for more information.) During this process you have to specify the Source language name and the Target language name. Adapt It uses these language names to build the project folder name.

The project folder contains an adaptation knowledge base and its backup; a glossing knowledge base and its backup if glossing is used; and a configuration file.
