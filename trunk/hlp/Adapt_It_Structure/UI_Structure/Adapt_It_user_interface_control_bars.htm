The Adapt It user interface control bars

The Adapt It user interface follows many of the Windows application standards. However, the unique features in Adapt It require special icons and tools. At the top of the interface are four control bars used to control and manage Adapt It’s features, although the Compose bar is generally hidden. The top two bars are typical of most Windows applications while the last two bars are unique to Adapt It.

Menu

The Menu control bar follows the basic organization of most Windows applications. This control bar contains File, Edit, View, Tools, Advanced, and Help menus. The Advanced menu is unique to Adapt It. In addition to these menus, Adapt It Unicode also has a Layout menu. To use any of these menus, click on the desired item and a drop-down menu appears.

Tools

The Tools control bar contains basic editing and printing icons contained in most Windows toolbars. There also a number of Adapt It specific icons as well.

Mode

The Mode bar contains several option buttons and checkboxes, along with a command button and a text box. The purpose of the Mode bar is to change temporarily your preferences as to how Adapt It should do its work.

Compose

Usually, Adapt It hides the Compose bar. This control bar provides you with a way to try out phrases as if you were using a “scratchpad.” When you are satisfied with the wording, you can transfer the material to the Phrase box, an Adapt It dialog box, or some other location.
