The Adapt It user interface

A user interface is what you as the user of an application sees as you work with the computer to do your task. Adapt It has its own unique interface, because it is a unique application. If you are familiar with other Windows applications you will recognize similarities to those programs. However, since Adapt It is a language adaptation application, it has many features not included in other applications.

The Adapt It user interface follows many of the standards for Windows application. However, in addition to the standard portions of the user interface, Adapt It contains a number of special structures, icons and tools. These unique features can be seen in the two main divisions of the interface:

  • Adapt It user interface control bars

    At the top of the interface are four bars used to control and manage Adapt It's features. Three bars are normally visible, with the fourth bar invisible until requested by the user. The control and management bars contain menus and Adapt It icons used to perform application functions.

  • Main window

    The Adapt It main window is below the control bars. The main window is where the most of the work is done. The purpose is to enter and manage text through the use of the Phrase box.
