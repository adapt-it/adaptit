Adaptation knowledge base

An adaptation knowledge base contains the related adaptations between the source language text and the target language text. When Adapt It encounters a hole, it searches the knowledge base for previous adaptation of a source language word or phrase. When it finds an adaptation, Adapt It inserts the adaptation into the target text portion of the hole and moves on to the next location.

It is possible that there may be two or more adaptation entries for a given source text word or phrase. When Adapt It encounters this situation, it relies on you to select to decide the best adaptation for the context. See Using the Choose Translation dialog box for more information.

When Adapt It cannot find an adaptation in the knowledge base, the application expects you to type in a new adaptation. When you press Enter, Adapt It stores the new adaptation in the knowledge base for future reference.

 Caution

  • The default action when Adapt It finds a source word or phrase, without an adaptation is to copy the source word or phrase into the Phrase box. If you press enter without making an adaptation, Adapt It accepts the source word or phase as an acceptable translation and stores it in the knowledge base. You can prevent source text from being copied to the phrase box by unchecking the Copy Source item in the View menu.

 Note

  • In these helps, unless stated otherwise the term knowledge base refers to an adaptation knowledge base.
