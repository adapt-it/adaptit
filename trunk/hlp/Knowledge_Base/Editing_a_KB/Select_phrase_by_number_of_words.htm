Select phrase by number of words

The knowledge base groups phrases by the number of source text words involved. The Adapt It Edit or Examine the Knowledge Base dialog box has a series of 10 tabs across the top, one for each set of phrases from one to ten words. For example to see all phrases that are five words in length, click on the 5 Words tab. The Source Phrase text box contains all of the qualifying phrases in alphabetic order.
