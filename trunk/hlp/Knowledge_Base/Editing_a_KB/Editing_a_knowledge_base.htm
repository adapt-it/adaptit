Editing a knowledge base

At times, you may need to make changes in the knowledge base. This process is done using the tool provided by Adapt It, and the user should not attempt to edit the knowledge base files outside of Adapt It.

To edit the knowledge base, do the following steps:

  1. Open the Knowledge Base Editor in one of the following ways.

    • Press Ctrl + K

    • Open the Knowledge Base Editor using the Tools menu.

    Adapt It opens the Edit or Examine the Knowledge Base dialog box.

    This dialog box is a powerful tool and requires an understanding of the knowledge base (For more information, see Edit or Examine the Knowledge Base dialog.)

  2. Select source phrase by number of words

  3. Select a specific knowledge base phrase.

  4. Select a knowledge base translation.

  5. Edit, add, or rearrange translations using the Edit and arrange command buttons.

  6. If desired, change the status of the Force Choice for this Item check box by clicking Toggle the Setting command button. See Force making a translation choice.

 Tip

  • If you select the source word or phrase before opening the knowledge base editor, Adapt It automatically finds the word or phrase as long as it is in the knowledge base.
