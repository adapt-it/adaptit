Select a knowledge base translation

For each phrase, the knowledge base contains at least one translation or a <no adaptation> entry. In some cases, the list box may contain both. The order of the translations depends on how they were entered. The user can rearrange the order as desired. To select a translation, highlight it by clicking the translation. That translation then appears in the Edit or Add a Translation text box for editing.
