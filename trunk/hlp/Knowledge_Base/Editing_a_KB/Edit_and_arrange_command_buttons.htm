Edit and arrange command buttons

The Edit and Arrange command buttons manage the changes made to the existing translations of a selected knowledge base (KB) phrase. All changes are made in the Edit or Add Translation text box and stored in the KB through the use of one of these command buttons. (See Using the Choose Translation dialog box for another way of managing and arranging multiple adaptations.)

  • Update command button

    After making changes to a translation, click the Update command button. The new translation replaces the previous translation in the knowledge base.

  • Add command button

    After entering a translation or editing an existing one, click the Add command button. The new or edited translation is added to the knowledge base. All previous translations are kept.

  • Remove command button

    Click the Remove command button to remove the highlighted translation in the Existing Translation list box. Adapt It displays a warning message box and deletes this translation from the knowledge base after you click Yes.

  • Move Up command button

    Click the Move Up command button to move the highlighted translation closer to the top of the Existing Translation list box. Adapt It automatically selects the first entry of the current adaptations. You can select any entry in the list and then use the button to move the entry.

    Move Down command button

    Click the Move Down command button to move the highlighted translation closer to the bottom of the Existing Translation list box. Adapt It automatically selects the first entry of the current adaptations. You can select any entry in the list and then use the button to move the entry.

  • Add <no adaptation> command button

    Click the Add <no adaptation> command button to add a <no adaptation> entry at the end of the list for the existing entries and into the knowledge base.
