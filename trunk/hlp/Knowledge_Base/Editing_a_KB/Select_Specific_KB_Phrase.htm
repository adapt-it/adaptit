Select a specific knowledge base phrase

The knowledge base groups phrases by the number of source text words involved. There may be many entries in the tabs associated with short phrases. As a user, you may find it difficult to locate a specific phrase when there are many possibilities. Adapt It provides a method to simplify the process. Simply types the first few letters of the desired phrase into the Type Key To Be Found text box. Adapt It finds and highlights the first phrase that matches those letters. You can type as many letters as are needed until the correct phrase appears in the Source Phrase list box. As an alternative, you user can also use the vertical scroll bar to show any entries that might be hidden and then click the desired phrase to select it.
