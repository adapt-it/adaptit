Removing knowledge base entries

At times incorrect adaptations may be entered into the knowledge base. The knowledge base can function correctly even with these faulty entries. However, these incorrect adaptations may be present in one or more of the adaptation documents. If you remove these entries from the knowledge base, there will be inconsistencies between the knowledge base and the documents. Adapt It can then take advantage of these inconsistencies to find and correct incorrect adaptations in the documents. See Consistency checking for more information.

You can remove knowledge base entries from two locations:

  • The Edit and arrange command buttons of the Edit or Examine the Knowledge Base dialog

  • Using the Choose Translation dialog
