Backup the knowledge base

Adapt It automatically backs up the knowledge bases whenever you close a project. You can prevent Adapt It from backing up the knowledge base. See Backups and KB tab.

You can also back up the knowledge base manually. To accomplish this, do the following steps:

  1. From the File menu select the Backup Knowledge Base option.

    Adapt It creates a copy of the knowledge base with the following file name format.

    • Name of project

    • The word “adaptations” followed by a dot (“period” or “full stop”).

    • BAK followed by a dot.

    • XML

    For example, if you had a “French to Spanish” project, the knowledge base backup file would be named French to Spanish adaptations.BAK.xml.

 Note

  • Adapt It performs the backup task without any visible sign to you.

  • The name of the working knowledge base for the above project would be French to Spanish adaptations.xml.

  • If the project is in Glossing mode, Adapt It makes a backup of the glossing knowledge base.
