Copying text to another location

To copy text to another location you must use the Clipboard. To do this, do the following steps:

  1. Select the text to be copied.

  2. Enter Ctrl+C.

    Adapt It copies the text into the clipboard.

  3. Navigate to the location where you want to copy the text.

  4. Position the cursor where the text should be copied.

  5. Enter Ctrl+V.
