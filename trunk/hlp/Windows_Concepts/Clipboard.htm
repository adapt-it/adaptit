Clipboard

The Microsoft Windows operating system maintains a special area in the memory of the computer. This area is called the Clipboard and can be used to store data temporarily. The data then can be retrieved for use in other applications, documents or files, and locations within the same document or file.

The clipboard works with the Edit menu functions, Tools bar icons, and shortcut keys:

  • Cut, , Ctrl+X -- Removes highlighted text from its location and stores it in the clipboard

  • Copy, , Ctrl+C -- Makes a copy of the highlighted text in the clipboard, but does not remove it from the document

  • Paste,, Ctrl+V -- Inserts the contents of the clipboard into text at the point marked by the cursor.

Adapt It uses the clipboard with a number of its dialog boxes and with the phrase box. In the main window, these functions work only within the Phrase box.
