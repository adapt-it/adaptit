Document dialog

The Document dialog allows you to select the document to work on next. You may want to open a previously adapted document or add an entirely new document to the project. The appearance of the dialog is affected by two conditions:

  • Is Book folder mode on?
  • Is <New Document > selected?

This example shows the dialog with Book folder mode ON and <New Document>selected.

     

    When Book folder mode  is On two things occur:

    • A message appears showing the folder where the current documents are being stored. In this case the folder is 2 John.
    • A command button Change Folder... appears allowing you to work with a different set of documents.

     

    At the bottom of the dialog there are one or two check boxes available depending on whether or not <New Document> is selected.

    • If <New Document> is not selected, only the check box labeled "Force UTF-8 encoding for source data input " appears.
    • If it is selected an additional check box labeled "Change USFM fixed space markers (~) to regular spaces" also appears.

     

     Tip

    • To change the Book folder mode you need to change the status of the Storing documents in Book Folders option in the Advanced menu.
