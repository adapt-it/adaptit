General Configuration file

The General Configuration file has the filename AI-BasicConfiguration.aic and is stored the Adapt It Work folder.

This file contains all application wide settings. Application wide settings apply to all Adapt It projects and documents. These settings serve as the default for each new project. Adapt It updates the file every time you close the application. When you launch Adapt It, the application applies the application-wide settings by using the General Configuration file.

Adapt It saves the file to the Adapt It Work folder when the application closes so that any changes are maintained.

Warning

  • Do not attempt to edit or change this file directly. Any changes to the preferences should be done inside of Adapt It.
