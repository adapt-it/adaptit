Project Configuration file

Adapt It builds a separate Project Configuration file for each project you create. This file is stored in the project folder and is named AI-ProjectConfiguration.aic. Each time you close a project or open a new document, Adapt It updates the project's configuration file. This file contains settings that are unique to projects, but also stores some of the settings that are in the General Configuration file. If the settings in the project configuration differ from the ones in the general configuration, the project settings take priority. Thus, each project can have separate fonts, text colors, and other settings. If you do not specifically establish these project settings, Adapt It uses settings from the general configuration.

Warning

  • Do not attempt to change these files directly. Use the preference tools in Adapt It.
