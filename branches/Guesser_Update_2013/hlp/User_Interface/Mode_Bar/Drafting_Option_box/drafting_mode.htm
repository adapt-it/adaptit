Drafting mode

Drafting mode is the default mode for Adapt It. In this mode, Adapt It operates in either in Automatic mode or Single Step mode. is the opposite of Drafting mode and moves the Phrase box to the immediate adjacent location each time Enter is pressed – whether that location had been adapted or not. By contrast Drafting mode skips over locations that have already been adapted. See Automatic mode and Single Step mode for further information.
