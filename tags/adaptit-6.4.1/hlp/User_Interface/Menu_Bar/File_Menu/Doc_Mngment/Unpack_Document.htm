Unpack Document

Unpack document combines with Pack document to assist translators in the process of sharing documents. You cannot unpack a document in Collaboration mode.

To use Unpack, do the following steps:

  1. Open the File menu and click the Unpack Document... option.

    Adapt It presents Load and Unpack The Compressed Document dialog.

  2. Navigate to the Packed document file and open it in one of the following ways:

    • Double click the name

    • Select it and then click Open

    Adapt It builds the necessary file structure, including constructing the appropriate project folder if it is not already present, decompresses the file and displays the document in the Main window.

 Note

  • Adapt It does not transfer a knowledge base in the pack and unpack process. In the unpack process, if no knowledge base exists for the project in the destination computer, Adapt It creates an empty one. Adapt It can start with an empty knowledge base to view and work on a document.

  • Packed files have a filename extension of .aip.

  • Packed document files from Adapt It are actually zip files with the filename extension of aip instead of zip. Normally there is no reason to do so, but it is possible to unzip the file outside of Adapt It, by changing the filename extension and using an appropriate application. Note however that the contents of the file include:

    • The adaptation document

    • The project configuration

    • Other information for setting up the required file structure.
