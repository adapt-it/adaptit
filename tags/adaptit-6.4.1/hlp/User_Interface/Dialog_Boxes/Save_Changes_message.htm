Save Changes message

This message appears when a document is closed after changes have been made. The options include:

  • Yes

  • This option saves the current document and clears the Main window. It makes any changes permanent by updating the hard disk knowledge base.

  • No

    This option closes the current document without saving any changes made in the last 10 minutes. These changes are lost. (See Auto saving tab for more information)

  • Cancel

    Adapt It does not do anything and returns to the document in the Main window.
