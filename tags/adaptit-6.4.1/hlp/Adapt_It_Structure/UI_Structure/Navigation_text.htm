Navigation text

Navigation text helps you to move around in a source text document.

This text contains the chapter and verse number of the source text. It also identifies the standard format markers as they appear in the document. Adapt It displays the navigation text, with its own color, as the top line of a strip in the main window.

Navigation text depends on the SFM markers included in the source text.

 Note

  • Navigation text has its own line in a strip, but is not part of a pile located below.

  • You can adjust the font and font color of navigation text in the Fonts tab.
