Source text preparation

The preparation of source text must be done outside of Adapt It. This can be done in other translation tools such as Paratext, Shoebox, and Toolbox. In addition, adaptations prepared in Adapt It can serve as source text for some other adaptation project.

Adapt It assumes the presence of SFM markers in the source text and uses them for organization and navigational purposes. Each of the above tools uses SFM and can provide them in the files to be used by Adapt It.

The source text should be as "clean" as possible. While it is possible to edit the source text and make minor corrections, it is best to have the source correct before using Adapt It.

It is possible to create source text files manually by using a text processor. This process is difficult, open to error, and requires a good knowledge of SFM
