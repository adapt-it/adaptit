Opening a Previously Created Project

To open a previously created project do one of the following:

  • Start Adapt It and click OK on the Welcome screen.

  • Close the current project if one is open. See Close a project. After closing the project, Click Start Working… in the File menu.

  • Click Start Working… from the File menu without closing the project

    If you choose this option, Adapt It displays the Document dialog, click <Back to display the Project dialog. Adapt It also closes the current document and ask you if you want to save any changes that have been made.

Adapt It displays the Project dialog and asks you to select a <New Project> or a previously created project.

To open a previously created project:

  1. Highlight the desired project.

  2. Click Next> to open the project.
