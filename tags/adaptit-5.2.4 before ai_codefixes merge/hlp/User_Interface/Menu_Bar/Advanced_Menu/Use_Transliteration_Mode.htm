Use Transliteration Mode

The Use Transliteration Mode entry is a toggle that turns Transliteration Mode on and off. This option is only available when SIL converters are installed with Adapt It Unicode. Although Adapt It (Regular) can support Transliteration, its type of project is not generally complex enough to justify its use.

Transliteration mode requires an extended discussion, thus we do not cover Transliteration and SIL in detail in these helps. However, the Adapt It installation files contain a document discussing these topics.

The use of Transliteration mode should be limited to projects specifically set up to use this application. Operating in Transliteration mode requires a special use of the Adapt It knowledge base. Adapt It allows the SIL converters to make the adaptation rather than the knowledge base. Therefore, as Adapt It find a hole it calls the SIL converter to make an adaptation and place it into the Phrase box. You need to determine if the adaptation is acceptable or not and do one of the following:

  • If the adaptation is acceptable, press either Ctrl+Enter or Alt+BackSpace.

    Adapt It places a <Not in KB> entry into the knowledge base.

  • If the adaptation is not acceptable, edit the entry in the Phrase box or enter a new adaptation. Then press Enter.

    Adapt It places the correct text into the knowledge base.

When Adapt It encounters the word or phrase the next time, it checks the knowledge base. If it finds a <Not in KV> marker, Adapt It calls the SIL converter to make the adaptation and then continues on to the next location. If the knowledge base does not have a <Not in KB> marker, Adapt It uses the adaptation in the knowledge base and then continues on to the next location.

 Note

  • For more information on Transliteration mode and SIL converters contact SILConverters.
