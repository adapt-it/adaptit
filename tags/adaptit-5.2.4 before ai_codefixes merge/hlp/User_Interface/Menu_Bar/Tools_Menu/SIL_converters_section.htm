SIL Converters section

The Tools menu contains a section that initiates the process of using SIL Converters.

There are only two options in this section, and they are not available if you have not installed the SIL Converter software. See SIL Converters overview for more information.

  • The SIL Converters… option allows you to start the process of selection selecting and running the appropriate converter for your language. The document SIL Converters in AdaptIt.doc, found in the Adapt It or Adapt It Unicode subfolder of the Program Files folder, explains the process in detail.

  • The Use SIL Converter option is a toggle that determines whether or not Adapt It uses the SIL Converter. It is available only when a converter has been loaded. If the option is checked, the converter is operational. If it is not checked, Adapt It does not use the converter.

 Note

  • When the converter is in use the Accept Changes Without Stopping option in the Consistent Changes section, becomes available. See the warning in the Accept Changes Without Stopping helps before using it.
