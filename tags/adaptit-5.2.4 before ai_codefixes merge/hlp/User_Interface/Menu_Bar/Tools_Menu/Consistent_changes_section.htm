Consistent changes section

Consistent Changes (CC) is a software application that has been built into Adapt It. However, Adapt It's adaptation process is not the CC software. CC is a potentially useful option, but most users may not need to use it.

This section of the Tools menu allows you to control the process of working with CC.

There are four options available to you:

  • Load Consistent Changes

  • Unload Consistent Changes

  • Use Consistent Changes

  • Accept Changes Without Stopping

 Note

  • Adapt It does not require the use of CC, but it is available for your use if you understand how to create CC rules. See the document CC.doc supplied with the Adapt It installation package.

  • CC is useful for making automatic well defined changes to the spelling of words when copied from the source text and before presented in the Phrase box. An example of a situation where this could be useful is where one language is used in two countries, but the spellings are different.

  • CC depends on using lists of carefully constructed rules. The process of developing these rules can be difficult since poorly defined rules can generate undesired results. SIL entities generally have consultants that can help you to define CC rules.
