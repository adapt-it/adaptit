Moving notes

The Moving notes section of the Edit menu, allows you to move the position of a note forward or backward one word or phrase at a time.

These options are available only when the document contains one or more notes and either the Phrase box is located to the immediate right of the Note icon. or the source text word or phrase to the immediate right of the icon is selected.

To move a note one word or phrase toward the end of the document

Do one of the following;

  • Enter Ctrl+3

  • Select the Move Note Forward option in the Edit menu.

To move a note one word or phrase toward the beginning of the document

Do one of the following;

  • Enter Ctrl+2

  • Select the Move Note Backward option in the Edit menu.

 Tip

  • If you need to edit source text and a Note falls within the selected text, you can temporarily move the note away from the selected text and then move it back after the changes have been made.

 Note

  • When you move a note, Adapt It moves the source text selection along with the note. This behavior allows you to quickly move the note additional times by pressing Ctrl+3 or Ctrl+2 repeatedly. This action takes place even if you started the process by placing the Phrase box instead of selecting source text.

  • When you move a note the Phrase box does not move with the note.
