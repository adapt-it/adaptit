Using Start Working… for Documents

To use the Start Working for opening or creating a document within a project the project must be open. See Opening a previously created project.

Once you have opened the proper project, do the following steps:

  1. Open the File menu.

  2. Click the Start Working… option.

    Adapt It opens the Document dialog box.

    See Document Dialog for more information.

    The dialog list box contains the option of <New Document> and the name all of the other documents in the project you have created if any.

  3. Choose one of the following options:

    • Click the What is a document? command button.

      Adapt It displays a message box explaining about documents. Click OK when you are done and choose one of the next two options.

    • Select the <New Document> item in the list box and click Next>.

      See Create a document using Start Working... for more information.

    • Select one of the previously created documents in list box and click Next>.

      Adapt It opens the document in the Main window.

 Note

  • You can alternatively use the New option of the File menu to create a document.

  • You can alternatively use the Open option of the File menu to open a document.
