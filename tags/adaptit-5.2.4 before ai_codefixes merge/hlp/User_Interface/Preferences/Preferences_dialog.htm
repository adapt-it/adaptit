Preferences dialog

Adapt It's preferences and settings can be made using the Preferences dialog. Access the Preferences dialog by clicking on Preferences… on the Edit menu.

The Preferences dialog initially appears with the Fonts tab selected. To set other preferences click on one of the other tabs at the top of the dialog. The Preferences dialog in the Unicode version allows for Right-to-Left language settings. See Unicode Preferences dialog.
