Fonts tab

The Fonts tab of the Preferences dialog box allows the user to change the font type, size, and color of the source text, target text, and navigation text. The accompanying illustration shows the format of the tab as presented in Adapt It Unicode.

The tab contains eight command buttons. Three of these buttons display the Font dialog where you can change the font type, the size of the text, and the basic color of the characters for the Source text, Target text, and Navigation text.

The remaining five command buttons allow more control over the color of the text for the three above types of text plus Retranslation text and Special text. Special text is text such as footnotes or editorial comments that have been inserted into the Source but is not part of the original document. See Managing font color on how to use them.

Adapt It Unicode allows the adaptation of languages that read right to left. In this case, the option boxes need to be clicked. The option boxes do not appear in the standard version of Adapt It.
