Printer

This section of the Print dialog box allows you to choose a printer, if you have more than one.

To choose from a list of available printers:

  1. Click the Down arrow in the Name: drop-down box.

  2. Highlight the printer you want to use and click on it.

The Properties command button allows you to change the options for the chosen printer. Each type of printer has its own set of properties and we cannot show them all here. Usually you can select the page orientation and page size.
