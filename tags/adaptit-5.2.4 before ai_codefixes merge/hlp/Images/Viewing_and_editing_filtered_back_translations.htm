Viewing and editing filtered back translations

Back translations are stored within Adapt It documents as filtered material. Filtered material is indicated on Adapt It's main window by small (usually green) wedge shaped icons.

You may click on these icons to view or edit any back translations stored in those locations. When you click on a filtered material wedge, the View Filtered Material dialog appears.

If the back translation (\bt) marker is not the first marker in the list at left, first select the back translation marker. The back translation text associated with the selected back translation (\bt) marker appears in the middle edit box. You can edit the back translation there. See View filtered material for more information.

 Note

  • The wedges containing hidden back translations are located just before the material to which the back translation applies, never after the material.
