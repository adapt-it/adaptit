Selecting source text

Many of the Adapt It procedures require you to select or highlight some portion of the source text. This can include merging or unmerging phrases, retranslating, and modifying adaptations.

To select source text, use one of the following techniques.

  • Click and drag

  1. Click and hold the mouse on the first source text word you want to highlight.

  2. Drag the pointer to the last text word you wish to select. You can move the pointer to the right, left, up, or down.

    Adapt It highlights the source text in yellow.

  • Click and Shift+Click

  1. Click on the first source text word you want to highlight.

  2. Shift+Click on the last text word you want to highlight. This word can be to the right or left of the first word. It can also be above or below the first word in the text.

    Adapt It highlights the source text in yellow.

  • Alt+Right Arrow or Alt+Left Arrow

  1. Place the Phrase box at the location below the first source text word.

  2. Press Alt+Right Arrow to select that word and the next word to the right.

  3. Continue pressing the Alt+Right Arrow until all you have selected all the text.

    Alt+Left Arrow works in the same way as Alt+Right Arrow but moves to the left instead of the right.

 Note

  • Because the selection process stops at any punctuation mark other than a comma, see Controlling punctuation boundaries to override the punctuation boundary restriction.
