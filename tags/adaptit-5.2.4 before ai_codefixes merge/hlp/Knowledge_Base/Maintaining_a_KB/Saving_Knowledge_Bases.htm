Saving knowledge bases

Knowledge bases may be saved in several ways:

  • Automatically

  • Manually

  • By editing the knowledge base

Automatic saving

Adapt It saves a knowledge base whenever a document is saved. Adapt It also saves the knowledge base every 15 minutes if the document has not been saved in that time. No action is required on your part.

Manually saving

You can also manually save the knowledge base.

Editing the knowledge base

Whenever you edit a knowledge base, your changes are saved automatically to the knowledge base.

 Note

  • No matter which way the knowledge base is saved, the saving process for the knowledge base does not make any changes to the adaptation document.

  • If you have made editing changes in the knowledge base, there may be some inconsistencies between the knowledge base and the adaptation document. These inconsistencies can be easily resolved by using Consistency Check… on the Edit menu.
