Load multiple changes tables

Adapt It Regular or Adapt It Unicode are designed to optionally handle up to four Consistent Change (CC) tables in series; that is, the output from one table becomes the input to the next. To do this, follow these steps:

  1. In the Tools menu select the Load Consistent Changes… option.

    Adapt It links itself to the CC software and opens the Get One Or More Consistent Change Tables dialog.

    There are four Table tabs allowing you to select up to four tables.

  2. In the list box for Table 1 tab, select the CC table you wish to use.

    To the right you will see the name of the table you have chosen.

  3. If you wish to use more than one table, select a new Table tab and repeat step 2 up to three time.

    In this way you can select up to four tables.

 Note

  • Sometimes it might be necessary to do an adaptation with more than four tables in series The Adapt It installation package includes a special CC table template that allows you to use more than four tables. This table called "table series as one.cct" is located in the Adapt It or Adapt It Unicode subfolders of the Program Files folder. This table allows you paste the rule contents of up to three tables into the appropriate places in the one table.

  • By using the template four times and saving each copy under a new name, you could have the equivalent of twelve sub-tables processed in sequence (four tables, each with three sub-tables).

  • The contents of this table could also be modified to make the table handle more than three sub-tables, thus allowing almost any number of CC tables to be used.

  • See the comments in the template table to learn how to use and modify it.
