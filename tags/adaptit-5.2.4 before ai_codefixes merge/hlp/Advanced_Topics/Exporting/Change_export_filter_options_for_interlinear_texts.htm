Change Export Filter/Options for interlinear (RTF) texts

This procedure allows you to control which standard format markers are included in an export of interlinear texts. (You can only export interlineally in RTF format). The default is to Export all markers and associated text including filtered text.

If you want to filter out (omit) certain markers and associated text during export:

  1. Click the Filter out selected markers and associated text from the exported document option.

    Adapt It makes the list box available for editing.

  1. Select a marker you want to change for the export.

  2. Depending on the buttons available, do one of the following:

    • Click Filter Out to omit the marker and its associated text from the exported text.

    • Click Include in Export to keep the marker and its associated text in the exported text.

    • Click Undo to restore any changes you have made.

  3. Make any adjustments wanted for the Special placement options for back translations, free translations and Adapt It notes exports area at the bottom of the dialog. See Placement of filtered material in interlinear RTF exports for more information.

  4. Click OK to accept your choice of options, or Cancel to abandon any export option changes.

    Adapt It returns to the main Export Document dialog where you can start (or cancel) the actual interlinear export process.

 Warning

  • Filtering of information from exports is not the same thing as filtering of information in the adaptation process. This topic is about filtering (omitting) markers and text information from exports. If you want to hide certain information during the adaptation process see Filtering of Information.

 Note

  • When no option boxes are checked any Free Translations, Back Translations or Adapt It Notes are formatted as footnotes.
