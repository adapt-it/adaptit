Interlinear Text Sample

This illustration shows how Microsoft Word represents interlinear text created using Adapt It's Export Interlinear Text command. Adapt It produces an interlinear RTF file which Microsoft Word can read and display. Adapt It maintains the alignment of text elements by placing them in tables with cells of appropriate size.

 Note

  • The picture above shows boxes around each cell. When you print the document, the boxes do not print unless you change the border format in Microsoft Word.
