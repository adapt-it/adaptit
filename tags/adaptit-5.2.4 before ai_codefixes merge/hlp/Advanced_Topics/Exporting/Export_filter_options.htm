Export Filter Options

Adapt It can export texts and documents for many uses. See Using exported texts and documents overview.

The options that are available at export time depend on the type of export you select, whether you want to export all markers or only selected markers, and (for rtf formatted exports) how you want any back translations, free translations, and notes displayed in those exported texts. See one of the following procedures to change the Export Filter/Options:

  • Change Export Filter/Options for source or target texts in standard format (SFM)

  • Change Export Filter/Options for source or target texts in rich text format (RTF)

  • Change Export Filter/Options for interlinear (RTF) texts

 Warning

  • Filtering of information from exports is not the same thing as filtering of information in the adaptation process. This topic is about filtering (omitting) markers and text information from exports. If you want to hide certain information during the adaptation process see Filtering of Information.
