Glossing

Glossing is a translation technique using a third language to assist the translation process. Glossing associates words and phrases in this third language (glosses) to source language words and phrases. Thus, with glossing, the translator has an alternate version of the text's original meaning. Adapt It allows any available glosses to be displayed along with the source and target language versions.

 Note

  • Glosses are done on a word-by-word basis.

  • In some cases, phrases may be glossed.

  • The grammar of the gloss follows that of the source language. If the glossing language and source language grammars differ significantly, the resulting glossing line may awkward to read.

  • When glossing mode is on, Adapt It stores and looks up glosses in a separate knowledge base.
