Open... (Get Source Text From Paratext)

When the project is in Collaboration mode, this option allows you to open document already in a project, but is available only when there is no current document visible in the Main window. To open a document, do the following steps:

  1. Make sure that there is no open document in the Main window. If there is an open document, close the document.

  2. Click on the File menu.

    The menu opens and the Open item should be usable and not "grayed out."

  3. Click Open... (Get Source Text From Paratext)

    Adapt It opens the Get Source Text from Paratext Projectdialog box. This box shows a listing of all of the current books and chapters in the project.

  4. Click on the desired book to highlight it.

  5. Click on a chapter to highlight it.

  6. Click OK.

    Adapt It closes the dialog box and copies the document into the Main window.

    The document is now ready for adaptation.

 Tip

  • The Open function can be started by pressing Ctrl-O or the Open icon  in the Tool bar if there is no document in the Main window .
