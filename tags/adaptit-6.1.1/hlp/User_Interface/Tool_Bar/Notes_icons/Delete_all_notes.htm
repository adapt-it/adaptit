Delete all notes

To delete all notes, do the following:

  1. Click the Delete All Notes icon.

    Adapt It displays a message box asking you to confirm that you want to delete all notes.

    • If you click Yes, Adapt It closes the message box and removes all the notes from the document.

    • If you click No, Adapt It closes the message box and nothing is changed.

 Warning

  • Once notes are deleted, there is no way other than retyping to restore them. Use the Delete button in the Note dialog box to delete a single note. See Delete a note.

 Note

  • If there are no notes in the document, the Delete All Notes icon is grayed out.
