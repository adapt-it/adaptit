Pack Document

The Pack command allows translators to easily share documents from one machine to another. To pack the open document, do the following steps:

  1. Open the File menu.

  2. Click the Pack Document… option.

    Adapt opens the Save As dialog window.

  3. Find the desired destination and change file name if necessary.

    • The Save in dropdown box allows you to choose where to save the document.

    • The File name dropdown box allows you change the document name. (You can omit the filename extension of aip, since Adapt It automatically adds it.)

  4. Click Save to create the file.

    Adapt It packs the file and stores it in the desired location.

 Note

  • The document to be packed must be open in the Adapt It Main window.

  • Only one document can be packed at a time.

  • You may want to save the file to a USB memory stick, a floppy diskette, or some other form of removable media so that the file can then be transferred to the destination computer. Files can also be sent as attachments to an email message.

  • Files packed in Non-Unicode Adapt It cannot be unpacked in Adapt It Unicode and vice versa.

  • You can pack documents in Collaboration mode but you cannot unpack documents.
