Adapt It document format (ADT)

Adapt It versions 2.4.1i and earlier use a binary file format for documents. Each document has its own file which you can identify by the filename of the document and the filename extension of "adt". Binary files such as adt files cannot be read by humans.

Versions 3 and greater of Adapt It use a standard format called XML as the default. These files have the name of the filename of the document with the filename extension of "xml". XML files can be read by humans. The current version of Adapt It can process either XML or ADT files.

 Warning

  • Do not try to edit an ADT file outside of Adapt It. The file cannot be read by word processors and you might accidentally corrupt the file.

  • Do not attempt to edit an XML file outside of Adapt It unless you are familiar with the format.
