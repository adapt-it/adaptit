Split a document

Splitting involves removing a portion at the start of a document and storing it in a document of its own; leaving the remaining part in the original document. You give the split off document its own name and you may rename the remaining document. The splitting process may take place repeatedly but it always take place at the start of the current document.

To split a document into smaller pieces, do the following:

  1. Determine where you want to split the document.

    If the document contains more than one chapter, you will want to split the document at the chapter boundaries. If the document has only one chapter, move your Phrase box to a logical division point such as a paragraph.

  2. Select the Split Document… option of the Tools menu.

    Adapt It displays the Split The Open Document into Smaller Documents dialog.

    The left hand list box shows you the name of all the documents currently in the folder. Use this box to inspect the document names; make sure that you do not use any of those names for your split off document. Adapt It will check the names and warn you if you make this mistake.

  3. Select the type of split you want to make by clicking one of the options in the Split At: option group:

    • The phrase box location

      This option splits the document at the current location of the Phrase box. Thus the pile where the Phrase box is located becomes the first pile of the remaining document. The split off document ends with the pile immediately before the Phrase box.

      Make sure that the Phrase box is located at a logical dividing point such as a paragraph.

    • A standard format chapter marker (\c)

      This option splits the document at the next occurrence of a chapter SFM marker (\c).

      This option is not available if there are no chapter markers.

    • Split into single chapter documents

      With this option, Adapt It searches the document and splits it into an additional document at each chapter marker (\c).

      This option is not available if there are no chapter markers.

  4. In the top text box at the right, type either:

    • The name of the portion to be removed

      The portion to be removed is the first part of the document up to the split point.

    • A generic name for the documents if you have chosen to split at each chapter marker.

      The name of the text box changes with this option Adapt It automatically adds chapter numbers to the end of the generic name. The chapter numbers include a lead zero where necessary to maintain the proper order in the filename lists. .

  5. You may want to type a new name for the remaining document into the lower text box.

    The lower text box disappears if you have chosen the Split into single chapter documents option.

  6. Click one of the command buttons at the bottom:

    • Locate Next Chapter

      Adapt It moves the Phrase box to the next chapter marker or displays a message saying that there are no more chapter markers.

    • Split Now

      Adapt It splits the document if it can. If it cannot, Adapt It displays one of several possible warning messages. Read the message and make corrections to allow the document to be split.

    • Close

      Adapt It returns to the Main window without splitting the document.
