Show or hide punctuation

Adapt It maintains two versions of the source and target texts. One version includes the punctuation and one does not. The Show/Hide Punctuation icons control this process.

To change from punctuation to no punctuation:

  1. Click on the Hide Punctuation icon.

    Adapt It displays the source and target texts without punctuation and changes the icon to.

To change from no punctuation to punctuation:

  1. Click on the Show Punctuation icon.

    Adapt It displays the source and target texts with punctuation and changes the icon to.

 Note

  • See Toggle number of source text lines or Toggle number of target text lines to see both the punctuation and no punctuation lines at the same time.
