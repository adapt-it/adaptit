Document management

This section of the File menu includes seven items. These items look a little different depending on whether or not collaboration with Paratext or BiblEdit is being used. There are the same items in each mode but some collabortation mode operations are not available.

The items may be available or not depending on what you are doing in Adapt It. If the item is dark you can use it. If the item is grayed out, you cannot use it at the current time. There is a slight differences in the two menus shown above. If you are operating in the traditional adaptation method, the menu appears as in the left column above. If you are working in collaboration mode you will see the entries as in the above right hand column.
