Book folders mode

In this mode Adapt It stores each document in a folder which is named the same as the name of a source text book. That folder is a subfolder in the Adaptations folder. In the standard situation these book names are the names of the books of the Bible -- both Old and New testaments, plus an extra folder called Other Texts for storing non-scripture adaptations. Book folders mode has to be explicitly turned on by the user; otherwise, Adapt It stores the documents directly in the project's Adaptations folder.

Use the Storing Documents in Book Folders option of the Advanced menu to turn Book folders on or off.

 Note

  • If you change the Book folders mode from off to on, Adapt It does not automatically move documents to the new folders. You need to use the Move document... option of the Tools menu to manually move documents. This option is available only if Book folders mode is on.

  • If the book subfolders have been created, and you turn Book folders mode off, you cannot see or use any of these folders until Book folders mode is restored. All documents remain in the appropriate book folder and are not moved to the Adaptations folder.

  • If you are adapting a document in Book folder mode and you turn Book folder mode off, Adapt It closes the document in the Main window and opens the Document dialog .

  • If Book folders mode is off, all new documents are stored in the Adaptations folder.
