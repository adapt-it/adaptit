Placement of Phrase-medial punctuation

If you make a phrase that includes punctuation, Adapt It displays a dialog box that allows you to include that punctuation in the appropriate places within the target text. Because your target text phrase may a different number of words than the source text phrase, Adapt It cannot determine where to place the punctuation. Thus, you need to determine where it should be placed.

To place the punctuation:

  1. Select the punctuation mark from the box at the left.

  2. Place the cursor at the proper location in the phrase in the box on the right.

  3. Click Place to insert the punctuation mark into the phrase.

    The placed punctuation is removed from the list, and the marks below are moved up in the list.

  4. Repeat the above steps as necessary.

  5. Click OK when you have placed all of the punctuation.

    Adapt It closes the dialog and places the content of text box into the Phrase box as the new adaptation.

 Warning

  • Be careful about including punctuation in a phrase. Do this only when the punctuation is required for every occurrence of the phrase.

 Tip

  • You can also manually type the punctuation directly into the text box and ignore the above procedure if you prefer.

 Note

  • It is not necessary to use any or all of the punctuation characters in the list box.

  • Adapt It does not include punctuation characters that are positioned immediately before or after the selected phrase.
