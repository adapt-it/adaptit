Adapt It projects, books, and documents overview

Adapt It organizes work at three levels-- projects, books, and documents.

Adapt It projects

 

A project is a collection of all the files necessary in the adaptation process of translating a set of texts in a source language into target language texts. Adapt It stores all projects in the Adapt It Work or the Adapt It Unicode Work folder. Each project has its own projects in a computer subfolder named usingfor the source and target language names. Adapt It projects are stored in the Adapt It Work folder as a subfolder. Each project subfolder has a unique name which uses the names of the source and target language. The project subfolder contains a project configuration file, an adaptation knowledge base, and a subfolder called Adaptations. If you are using glossing, the project folder also includes a glossing knowledge base. The Adaptations subfolder contains the document files directly or within Book subfolders.

Adapt It books

The book structure is optional and consists of a series of subfolders within a project. Book folders are stored in the Adaptation folder of the project. Each book folder can contain one or more Adapt It documents.

A book folder represents a logical unit of the total project. For example, when translating the Bible, each Adapt It book folder contains documents related to a specific book of the Bible.

If you decided to use Book folders mode, Adapt It can automatically build the subfolders for Bible books. However, the book structure can be used for other collections of documents. In order to use the book structure with non-biblical books, you will have to modify one of the configuration files. Contact the authors for information on how to do this.

Adapt It documents

Documents are where the actual work of adaptation takes place. A document contains, at a minimum, the source text and the target text to the extent it has been adapted. The document can also contain other adaptation helps such as: gloss text, back translations, free translations, and notes.

Adapt It documents are stored in the Adaptations subfolder of the project. If you are using Book folders mode, the documents are stored in the appropriate book folder.
