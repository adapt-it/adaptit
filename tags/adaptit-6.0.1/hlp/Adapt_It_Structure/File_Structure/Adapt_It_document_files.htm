Adapt It document files

Adapt It supports a number of file formats for documents. Each file format serves a special purpose.

Adaptation documents

The current versions of Adapt It use two forms of XML documents for purposes of adaptation. The form used by Adapt It (Regular) is different from that used by Adapt It Unicode . The two forms are not compatible.

Legacy documents

Legacy versions of Adapt It used binary files. These binary files are no longer used by Adapt It. Adapt It Regular can still read them but converts them to XML format. Adapt It Regular uses a version of XML that supports the fonts used in the original binary files.

Input documents

Adapt It (both versions) only accepts Standard Format Text Files as source inputs for adaptation.

Export documents

Adapt It exports documents using one of three forms:

  • Standard Format Text Files

  • Rich text format

  • Open XML for Editing Scripture (OXES) files
