Edit, add, or rearrange translations

Once you select a translation and it appears in the Edit or Add Translation text box, you can then edit the translation, type in a new translation, delete the current translation, or move the translation up or down in the order of possible translations. In addition, you can add an <no adaptation> entry.

For information on using these buttons see: Edit and arrange command buttons
