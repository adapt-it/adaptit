Single Step mode process

Single Step mode allows you to approve or modify every adaptation done by Adapt It before the Phrase box moves to a new location. In single step mode, Adapt It:

  1. Stops at each empty location

  2. Inserts an adaptation from the knowledge base if available

  3. Waits for you to do one of the following:

    • Enter a new adaptation if there is none. (You can adapt a single word or Make a phrase if desired.) and press Enter

    • Change the current adaptation and press Enter

    • Press Enter to accept the current adaptation.

  4. Updates the knowledge base with current adaptation and moves to the next empty location.

Adapt It continues the process until the end of the source text is reached.

 Note

  • Single Step mode is the opposite of Automatic mode.

  • Automatic mode and single step mode are available only in Drafting mode.

  • Some translators prefer Single Step mode because it gives more immediate control over developing the meaning of the target text.
