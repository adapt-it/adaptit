Print Setup

Print Setup provides a way to set up the default setup for the printer you want to use with the project or document. To access Print Setup do the following.

  1. Click on the File menu on the Menu bar.

  2. Click on the Print Setup option.

    Adapt It opens the Print Setup dialog box.

    To make changes see Using the Print Setup dialog.

  3. When you have made the changes, click OK to finish or Cancel to ignore any changes.

    Adapt It returns to the Main window.
