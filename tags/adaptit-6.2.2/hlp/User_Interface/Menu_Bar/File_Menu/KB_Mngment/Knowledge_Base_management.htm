Knowledge base management

Generally the Adapt It knowledge bases operate without you needing to work with them directly. Nevertheless, at times you may need to make adjustments and corrections in the knowledge base. Only one knowledge base is active at any one time. Normally, the adaptation knowledge base is active. However, in glossing mode the glossing knowledge base becomes active. Any corrections or adjustments are applied only to the active knowledge base.

The Knowledge base (KB) is controlled using the File menu.

There are three options in this section of the File menu:

  • Save Knowledge Base

    Adapt It saves any changes you have made to the KB.

  • Backup Knowledge Base

    Adapt It makes a copy of the current KB and saves it into a BAK file

  • Restore the knowledge base

    This option is available only when no document is open in the project. Close the current document by using the Close option of the File menu.
