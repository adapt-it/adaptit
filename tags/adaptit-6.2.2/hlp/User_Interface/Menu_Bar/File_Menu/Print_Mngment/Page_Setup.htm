Page Setup

This option allows you to control the margins of the page to be printed.

To obtain the result, do the following steps:

  1. Open the File menu on the Menu bar.

  2. Click the Page Setup option.

    Adapt It opens the Page Setup dialog.

    This dialog allows you to set the Margins for the output documents. To make these changes see Using the Page Setup dialog.

  3. Click OK to accept your changes.
