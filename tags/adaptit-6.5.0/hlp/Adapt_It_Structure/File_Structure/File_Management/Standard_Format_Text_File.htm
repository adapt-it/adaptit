Standard format text file

A standard format text file is a variation of plain text files. It includes Standard Format Markers and can be used as a source text file to Adapt It or other translation applications such as Paratext.

 Note

Plain text files usually have a filename extension of txt. However, applications such as Paratext may use their own filename extension. For example, Paratext uses the filename extension sfm. Adapt It can read any plain text file regardless of filename extension.
