Enable punctuation copy

Adapt It's default behavior is to copy punctuation from the source text into the target text as the Phrase box moves to a new location. Adapt It does allow you to turn off this behavior temporarily for one location at a time. See No punctuation copy for how to turn it off.

The Enable punctuation copy button is the toggle button to the No punctuation copy button and appears only when Adapt It is in the no copy mode. This mode applies only to the current location of the Phrase box. When you move the Phrase box the No punctuation copy button reappears.

If you decide to return to the punctuation copy mode before moving the Phrase box, do the following:

  1. Click the Enable punctuation copy button.

    Adapt It restores the No punctuation copy button.

Note

  • The only time you should need to use this button is when you change your mind about using No punctuation copy mode.
