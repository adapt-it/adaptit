Help File updates

This document lists the recent changes to the Help files in our work to make them reflect the current version of Adapt It. You can check on the status of the changes to documentation as they occur by consulting this page.

 

Changes are listed in a batch listed by date. The most recent batch is stated at the front of the document. This page can be accessed by from:

  • Using Adapt It
  • Help menu

Changes as of March 21, 2011 include:

This set of changes leaves the Help files for pre-6.0 Adapt It largely intact. This is the beginning of the changes to bring the files to conform with version 6.0. The major items that have been affected include:

  1. Numerous images have been changed to reflect the 6.0 user interface. However, there remain many images that have not been updated. In most cases you should be able to relate the old image to what you see on the screen.
  2. The Menus have been slightly revised and new menu Export-Import has been added. A number of the options from the File menu have been moved there.
  3. Updates have been made in the following areas:
    • Adapt It Concepts
    • Adapt It Structure
    • Getting Started
  4. New features that have not yet been addressed:
    • Administration process
    • Admin menu
    • LIFT files definition and purpose, they are mentioned as appropriate  in topics about the Files used by Adapt It
    • OXES files, these will not be incorporated until version 6.1.
