Jump to the next note

To jump to the next note, do the following:

  1. Check that your Phrase box is located before at least one note.

  2. Click the Jump to the next note button.

    The Note dialog appears.

  3. Make any changes to note in the dialog's edit box. If the note is longer than the note dialog can display at one time, use the dialog's scroll bar to view the entire note. See Using the Note dialog for more information.

 Note

  • If there are no notes in the document, the Jump to next note button is grayed out.

  • If the Jump to next note button is grayed out and there are notes in the document, you have probably selected a source text word or phrase. Click in the target text to open the Phrase box or click anywhere in the white space. This deselects the source text and activates the button.

  • If the Phrase box is located after of all the notes, Adapt It has no more notes to open and nothing happens.
