Export Interlinear Text

In Adapt It you can export both the source text and the target text into an Interlinear Text document. This document resembles the Adapt It Main window in format. (See Interlinear Text Samples).

To export the interlinear text, do the following steps:

  1. Click on the File menu.

    The menu opens and the Export Interlinear Text item should be usable and not "grayed out."

    If the item is not usable, there is no document in the Main Window. You need to Open a document before proceeding.

  2. Click Export Interlinear Text

    Adapt It opens the Export RTF Interlinear Document dialog.

    This dialog allows you to choose various options for the output document. Interlinear Text can only be in RTF format. (See Using the Export RTF Interlinear Document dialog for more information.).

  3. When you are done selection options, do one of the following:

    • Click OK to produce the document. Adapt It opens a Save As dialog box. In the File Name text box, it suggests a filename based on the current document name.

    • Click Cancel to return to the Main Window.

    • Click Export Filter/Options to open the Export Filter/Options dialog for interlinear texts.

      The Export Filter/Options dialog allows you to control the suppression of standard format markers and the placement of certain information in the RTF document.

      After you close the Export Filter/Options dialog, Adapt It returns you to the current dialog and you have to select one of the two options above to continue.

 Note

  • The Gloss Text row is only available if the See Glossing option is selected/checked on the Advanced menu.

  • The export of RTF Interlinear Documents can create very large files. To avoid such large files you may wish to limit the Output Range. The most common way to limit the range is to select the Chapter/Verse Range option. When this button is selected, the from chapter and from verse edit boxes, and the to chapter and to verse text boxes are activated.

  • Export RTF Interlinear Document processes all standard format markers and their associated text, including any filtered material. If you do not want to export everything, click on the Export Filter/Options button to specify what to exclude from exports. See Export Filter/Options.
