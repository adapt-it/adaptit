Active location

In Adapt It, each document has many storage locations. In the Main window, the visual representation of the storage location is called a pile. The pile shows the current source text, its adaptation if any, and potentially a gloss for the storage location. Only one storage location is available for work at a time. This location is called the active location and is where the Phrase box is located. All changes to the active location are done within the Phrase box.

In Drafting mode Adapt It automatically moves the active location to the next available hole after you press Enter. In Reviewing mode, Adapt It moves the phrase box to immediate next pile, making it the new active location--whether or not that pile already has an adaption. The application does not make a knowledge base lookup in reviewing mode.
