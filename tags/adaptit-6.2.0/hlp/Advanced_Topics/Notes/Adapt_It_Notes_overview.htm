Adapt It Notes overview

The Adapt It Notes feature makes it possible to store consultant notes, advisor notes and exegetical notes within Adapt It documents. Since notes are stored as filtered information they remain hidden within Adapt It until you want to see them. Like other kinds of filtered information, notes are retained and passed along by default whenever you export your documents as standard format files. Usually the notes that you or an advisor or a consultant add to your document would also be helpful to a translator who uses your target text as a source text for a related language adaptation project or in a cluster language project.

Adapt It makes it easy to handle notes through the use of the following procedures:

  • Add a note

  • View a note

  • Edit a note

  • Delete a note

  • Navigate through the notes in a document

  • Move a note

 Caution

  • There can be a problem when propagating Notes and other filtered material. This type of material can block the ability of Adapt It to make a phrase if it crosses the boundary of a filtered section. In such cases, you may need to delete the filtered material to make the phrase. With Notes, you can move the note to some other location, go back and make the merger and then move the note back to its proper location.
