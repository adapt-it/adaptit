Preparing knowledge base entries in a word processor

Knowledge base (KB) entries are generally created as a normal part of the adaptation process. However, you can prepare knowledge base entries by using a word processor or Microsoft Notepad.

To prepare entries, do the following steps:

  1. For each source text word or phrase:

    • Type \lx

    • Leave a space

    • type the source text word or phrase

  2. For each associated target text work or phrase:

    • Start a new line

    • Type \ge

    • Leave a space

    • Type the target text word or phrase

    • Repeat step 2 until all associated target words or phrases are entered

  3. Insert a blank line

  4. Repeat steps 1 through 3 until all source and associated target words or phrases have been entered.

  5. Save the file as a txt file.

  6. Import the entries into the KB.

 Note

  • Adapt It only recognizes txt files for importing knowledge base entries. If you use a word processor to create the file, be sure you save the file as a text file.

  • Each word or phrase must have its own line.

  • A blank line between each entry (the combination of source word or phrase and the associated target words or phrases) is optional but makes the entries easier for a human to read.

  • Only the \ge and \lx SFM are recognized for the KB. Adapt It ignores any other markers and associated data in an KB import file.
