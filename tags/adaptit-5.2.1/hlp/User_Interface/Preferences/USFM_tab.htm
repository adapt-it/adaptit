See Set USFM Preferences for information on using this tab.

 Note

  • In these helps and in Adapt It itself, SFM or (sfm) is an abbreviation for "Standard Format Markers" and USFM is an abbreviation for "Unified Standard Format Markers." Both types of markers begin with a backslash (\). The markers are often found in the source text used for constructing Adapt It documents. They were placed in the source text in order to guide software used for publishing; however, Adapt It also uses a number of them for its own purposes.

  • Changes involving the Standard Format Marker (SFM) set should be made here in the USFM page before making any Filtering changes in the Filtering tab.
