Using the Choose Translation dialog

The Choose Translation dialog allows you the ability to select an adaptation for the current location of the Phrase box. This dialog box appears under one of the following conditions:

  • Adapt It determines that there are multiple adaptations in the knowledge base for the source text word or phrase.

  • Adapt It determines that the Force Choice for This Item check box has been set for this source text word or phrase. See Using the Force Choice For This Item checkbox.

  • You open the dialog box by using the Show the Choose Translation Dialog icon on the Tool bar. See Show the Choose Translation dialog.

The dialog contains:

  • Translations which are currently in the knowledge base list box.

    This box lists all of the current translations. Adapt It highlights the top item in the list. Click OK to place it in the Phrase box. Otherwise, click on one of the other translations to select it.

    Normally, you will want the most used translation as the first entry in the Translations list box. Use the Move Up command button to accomplish this.

  • New Translation text box

    If you want to add a new translation to the knowledge base and insert it into the Phrase box, type the translation into this box.

  • References: text box

    This text box tells you how many times the selected translation is used. The value changes as you select different translations. Normally, you will want the most used translation as the first entry in the Translations list box. Use the Move Up command button to accomplish this.

  • Move Up command button

    Click this button to move the selected translation one position up in the list. This button is not available when you have selected the translation at the top of the list.

  • Move Down command button

    Click this button to move the selected translation one position down in the list. This button is not available when you have selected the translation at the bottom of the list.

  • Remove from KB command button

    Click to remove the selected translation from the Knowledge Base.

  • OK command button

    Click OK to insert the selected translation or the contents of the New Translation text box into the Phrase box. Adapt It closes the dialog box.

  • Cancel command button

    Click Cancel to ignore any changes and to close the dialog box.

 Note

  • You will want to place the most used translations at the top of the Translation list box. Use the Move Up and Move Down buttons to rearrange the sequence of items. The Reference text box can help you to determine the most used translations.
