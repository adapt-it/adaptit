Filtering tab

Preferences for filtering (or hiding) of information can be set using the Filtering tab of the Edit menu's Preferences dialog, which looks like this:

The Filtering page displays three groups of settings; settings for the current document at the top, default settings for the current project in the middle, and default factory settings at the bottom. You can change the current document's settings and the current project defaults, but you cannot change the factory defaults, because they are read only.

 Warning

  • Be sure that you have made any necessary changes in the USFM tab before making changes here.

 Note

Normally the filtering settings will be the same for all documents in your project, so This Document's Filter Settings list of markers (the top list) should have the same markers checked for filtering as the markers in the Project Filter Defaults list (the middle list). You may make the current document's filtering settings different if you wish, but you should eventually insure that all documents in a project use the same filtering settings that the overall project uses.
