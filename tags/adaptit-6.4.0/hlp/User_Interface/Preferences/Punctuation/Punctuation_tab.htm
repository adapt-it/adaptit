Punctuation tab

The Punctuation tab of the Preferences dialog box allows you to add, change, or delete punctuation marks in the Punctuation Correspondence table.

To open the tab, do the following:

  1. Open the Edit menu.

  2. Click Preferences.

  3. Click the Punctuation tab.

See Managing Punctuation for information on how to use this dialog tab.

 Note

  • You must scroll down the dialog box to see all of the fields.

  • This tab works in the same way as the Punctuation dialog.
