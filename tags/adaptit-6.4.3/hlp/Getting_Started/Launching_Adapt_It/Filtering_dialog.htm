Filtering dialog

The filtering dialog appears as part of the process of creating a new project. This dialog allows you to set the project's filtering conditions. You can override Adapt It's default (Factory Filter) settings for the purpose of your project. Any changes you make in the defaults are applied to each new document as it is created in the project.

This dialog works like the Filtering tab of the Preferences dialog. However, since you have not yet created any documents for the project, you cannot set document filter settings. In addition, you cannot change Adapt It's default (Factory) filter settings.

To change the project filter settings see Set filtering preferences.

After making any desired changes, do one of the following options:

  • Click Next to move to open the Document dialog.

  • Click Back to return to USFM dialog.

  • Click Cancel to close the project and leave the Main window empty.

  • Click Help to show these helps.
