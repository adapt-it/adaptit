Using Start Working for Projects

To use the Start Working… option, do the following steps:

  1. Open the File menu.

  2. Click the Start Working… option.

    Adapt It opens the Project dialog box

    The dialog list box contains the option of <New Project> and the name all of the other projects you have created if any.

  3. Choose one of the following options:

    • Click the What is a project? command button.

      Adapt It displays a message box explaining about projects. Click OK when you are done and choose one of the next two options.

    • Select the <New Project> item in the list box and click Next>.

      Adapt It starts the The Start Working... wizard and leads you through the process of creating a new project.

    • Select one of the previously created documents in list box and click Next>.

      Adapt It opens the Document dialog box and allows you to select a previously created document or to create a new one. See Document dialog on using this dialog.
