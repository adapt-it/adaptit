Save As...

This option is a control to save the current adaptation document with a new filename or to save it in the prior XML format. You can use Save As... only in traditional adaptation mode. Collaboration mode does not allow this option to be used.

Adapt It versions prior to 6.0 used a specific XML format. The XML format has been revised to meet the needs of version 6.0 and  is the preferred format. See XML in Adapt It .

Even if you choose to save a document in the legacy format, Adapt It now only uses the new format. Thus, when you read the saved document in for the next session, Adapt It converts the document back to the new format.

 Note

  • This option does not allow you to save the document in a different folder.

  • If you save the document in the legacy version and then save the file again normally, either manually or automatically, Adapt It converts the file back to the preferred format . Adapt It can open either type of file but saves it according to the current setting. Adapt It maintains only one copy of the adaptation file, based on the setting when it was last saved.

Tip

  • If you need to have a copy of document file in legacy XML format, for sharing with other projects:
    1. Use Save As... and select the Legacy XML option.
    2. Click Save
    3. Close the document but do not resave the document.
    4. Use your operating system commands to save a copy of the file to a new directory.
