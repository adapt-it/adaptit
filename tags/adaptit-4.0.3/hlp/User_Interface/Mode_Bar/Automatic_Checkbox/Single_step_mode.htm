Single Step mode

Single Step mode moves the Phrase box to the next empty location and inserts an adaptation if one exists in the knowledge base. Adapt It then stops and waits for you to change the adaptation or to press Enter.

 Note

  • Single Step mode is the opposite of Automatic mode. To change from Automatic mode to Single Step mode, click the Automatic mode checkbox.

  • Single Step mode is available only when the Drafting option button is selected.
