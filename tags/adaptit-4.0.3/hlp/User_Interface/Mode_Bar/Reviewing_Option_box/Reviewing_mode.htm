Reviewing mode

Reviewing mode provides a method to examine each adaptation in a document. In this mode Adapt It move the phrase box one location at time and displays the current adaptation. Reviewing mode is the opposite of Drafting mode. Drafting mode skips over any locations that have been already adapted. Reviewing mode does not allow the use of Automatic mode nor Single Step mode.

 Tip

  • The most effective use of Reviewing mode to help your reading of the target text. In reviewing mode, the Phrase box move one location at a time; therefore you can maintain the context of reading. By contrast, Drafting mode moves to the next available hole which may be many locations ahead.

 Note

  • In reviewing mode, Adapt It does not do a knowledge base lookup as it moves the Phrase box. Thus, if it encounters a hole it will not supply an adaptation. You can type one in or check the knowledge base yourself by clicking the Show The Choose Translation Dialog button.

  • In reviewing mode, Adapt It does not allow you to move the Phrase box into a retranslated area of the document. Therefore, you must do one of the following:

    • Manually move the Phrase box past the retranslation area.

    • Select one or more words of the retranslation's source text. Adapt It then allows you to edit or delete the retranslation.
