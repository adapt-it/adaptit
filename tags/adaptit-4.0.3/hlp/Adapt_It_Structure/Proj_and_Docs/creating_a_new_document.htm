Creating a new document

Adapt It provides two starting points to create a new document:

  • Creating a document using Start Working

    This option is the standard way of creating a document in that it is part of the startup process.

  • Creating a document using New

    This option allows you to create a new document in the same project and folders as the current document.

Whatever the starting point, Adapt It opens the Document dialog box.

Use the following procedure to create a new document.

  1. Select <New Document> item in the list box.

  2. Click Finish

    Adapt It opens the Input Text File for Adaptation dialog box.

  3. Find the source file for the new document.

    Move to another folder if necessary.

  4. Click on the desired file name.

  5. Click Open.

    Adapt It opens the Type Output Document Name dialog box. The application takes the name of the source file as a suggested name. The user can change it to another name if desired.

  6. Click OK.

    Adapt It accepts the name, creates a new Adapt It document file and opens the new document in the workspace.

 Note

  • A new document can be created only when the Adapt It workspace is empty. Adapt It closes any active documents before creating a new one.

  • Adapt It creates the new document in the current project's work folder. If the project is in Book Folder mode, the document is created in the current Book Folder.

  • If you wish to create a new document for a different project, the current project must be closed and the other project opened.

  • A project needs to be open in order to create a document

  • Adapt It can deal with only one document at a time, thus it has to close any open document before it creates a new one.
