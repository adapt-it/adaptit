Adapt It document files

Adapt It supports a number of file formats for documents. Each of these file formats are for specific purposes.

Adaptation documents

The current versions of Adapt It use two forms of XML documents for purposes of adaptation. The form used by Adapt It (Regular) is different from that used by Adapt It Unicode. The two forms are not compatible. For more information see XML in Adapt It.

Legacy documents

Legacy versions of Adapt It used binary files. See Adapt It document format

Input documents

Adapt It only accepts Standard Format Text Files as source inputs for adaptation.

Export documents

Adapt It exports documents using one of two forms:

  • Standard Format Text Files

  • Rich text format
